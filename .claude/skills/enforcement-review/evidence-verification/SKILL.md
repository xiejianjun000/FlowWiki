# 证据链闭环核验 Skill

## 触发条件
需验证案卷证据是否形成闭环

## 输入/输出契约
- 输入：案卷证据清单
- 输出：证据链闭环结论（印证关系图 + 断点清单）

## 执行步骤
1. 列证据清单（主体/违法事实/定性/危害后果/裁量证据）
2. 先过采样关 → 若含监测数据调用采样合法性审查
3. 核单一依赖 → 是否有唯一证据定案
4. 核印证关系 → 时间/地点/数量一致性
5. 输出断点清单 + 印证关系图
