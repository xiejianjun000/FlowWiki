# Change Assessment Skill — 变更评估

## 功能

评估排污许可证变更事项的合规性和影响范围，判断是否需要重新申请、变更或重新核发。

## 输入

```json
{
  "permit_id": "许可证编号",
  "change_type": "新增|改造|拆除|其他",
  "change_content": "变更内容描述",
  "current_permit": "当前许可证信息"
}
```

## 输出

```json
{
  "status": "需重新申请|需变更|无需变更",
  "impact_areas": ["排放量", "排放浓度", "排放口位置", "污染防治设施"],
  "legal_basis": ["引用条款1", "引用条款2"],
  "required_materials": ["所需材料列表"],
  "summary": "评估结论"
}
```

## 评估要点

1. 变更内容是否影响许可事项
2. 排放变化是否超过阈值
3. 是否触发重新申请条件
4. 变更流程是否合规

## 约束

- 引用排污许可管理条例具体条款
- 区分行政许可事项变更和非许可事项变更
- 给出明确的变更路径建议
