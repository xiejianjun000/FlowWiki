# Policy Monitor Skill

> 政策监控 — 监控法规、标准、政策的更新动态

## 功能

监控法规、标准、政策的更新动态，评估对企业合规状态的影响。

## 输入

```json
{
  "scope": "监控范围",
  "entity": "企业名称（可选）",
  "period": "监控周期"
}
```

## 输出

```json
{
  "updates": [
    {
      "type": "法规|标准|政策",
      "title": "更新标题",
      "change": "变更内容",
      "effective_date": "生效日期",
      "impact": "high|medium|low"
    }
  ],
  "summary": "监控概况"
}
```

## 监控范围

1. **法律法规**：新颁布、修订、废止
2. **标准规范**：新发布、修订、废止
3. **政策文件**：通知、指导意见、规划

## 约束

- 监控来源需权威可靠
- 影响评估需结合企业实际情况
- 定期更新监控范围
