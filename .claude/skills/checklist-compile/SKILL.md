# Checklist Compile Skill

> 清单编制 — 根据企业信息生成合规检查清单

## 功能

根据企业行业类型和合规要求，生成结构化的合规检查清单。

## 输入

```json
{
  "entity": "企业名称",
  "industry": "行业类型",
  "audit_type": "审计类型",
  "scope": "检查范围"
}
```

## 输出

```json
{
  "checklist": [
    {
      "category": "检查类别",
      "items": [
        {"item": "检查项", "standard": "依据标准", "method": "检查方法", "required_doc": "所需文件"}
      ]
    }
  ],
  "total_items": 42
}
```

## 检查类别

1. 资质证照（营业执照、许可证、资质证书）
2. 内控制度（管理制度、操作规程）
3. 人员管理（资质、培训、健康）
4. 安全生产（设施、记录、应急）
5. 数据合规（数据安全、隐私保护）

## 约束

- 清单需覆盖企业主营业务
- 检查项需引用具体法规/标准
- 清单需定期更新
