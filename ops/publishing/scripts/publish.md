# 掘金/知乎发布操作指令

## 发布到掘金

### 前置条件
- 掘金账号已绑定 GitHub（github.com/xiejianjun000）
- agent-browser 技能已加载

### 操作步骤

1. **加载浏览器技能**
   ```
   @agent-browser 打开掘金创作者中心
   ```

2. **登录**
   - 掘金首页 → 右上角"登录" → GitHub 图标登录
   - 授权后自动跳回

3. **创建文章**
   - 点击"写文章"
   - 标题：`Karpathy 提出了 LLM Wiki 的构想，我把 6 个致命缺口全补上了——FlowWiki 开源首发`
   - 粘贴 `juejin.md` 正文

4. **上传配图**
   - 封面：`assets/cover.png`（800×420）
   - 插图依次上传到对应位置

5. **设置标签**
   - `人工智能` `开源` `知识管理` `LLM` `GitHub`

6. **设置摘要**
   ```
   让知识像代码一样编译、缓存、复利增长。不是又一个 RAG，而是一个带防幻觉机制的 AI 知识库编译器。
   ```

7. **预览 → 发布**

---

## 发布到知乎

### 操作步骤

1. **创建文章**
   - 知乎 → "写文章"
   - 粘贴 `zhihu.md` 正文（开头不要"H1"，知乎会自动加）

2. **添加话题标签**
   - `人工智能` `开源项目` `知识管理`

3. **设置封面**（知乎不支持自定义封面，用文中第一张图）

4. **预览 → 发布**

---
