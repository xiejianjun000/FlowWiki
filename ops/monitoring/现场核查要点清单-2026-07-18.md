## 三、现场核查要点

### 🔴 A区：自动监测异常专项（线索1）

#### A1. 工况标记真实性核查（18次标记 100%关联超标）

**法规依据：《大气污染防治法》第24条；HJ 355《污染物在线监控规范》**

- [ ] **A1-1. DCS曲线专项调取**
  - 要求企业提供2026年1月1日—2月24日完整DCS历史曲线
  - 重点调取参数：炉膛温度、蒸汽流量、引风机电流、脱硫系统启停信号
  - 逐一核查18次工况标记（停炉8+启炉10）的起止时间
  - **发现项：若DCS显示启炉时长<实际标记时长，或温度未降至停炉阈值即已标记，涉嫌虚标**

- [ ] **A1-2. 启炉/停炉时长合理性判断**
  | 参数 | 正常范围 | 异常判断 |
  |------|---------|---------|
  | 启炉时长 | 通常1-4小时（回转窑规格不同） | 标记时长超过实际需要 → 刻意延长覆盖超标时段 |
  | 停炉时长 | 通常0.5-2小时 | 同上 |
  | 炉膛温度阈值 | 参考企业工艺参数 | DCS未记录降至常温即标记停炉 → 提前标记 |

- [ ] **A1-3. 正常工况数据剥离核查**
  - 将18次工况标记时段从总数据中剔除
  - **单独提取"正常工况"时段的SO₂/颗粒物/NOₓ数据**
  - 若正常工况存在未标记的超标 → 涉嫌故意漏标 → 监测数据造假
  - 若正常工况无超标，但标记时段100%覆盖所有超标 → 涉嫌"以工况标记掩盖设施低效问题"

#### A2. 在线监测设备合规性核查

**法规依据：《大气污染防治法》第25条；《污染源自动监控管理办法》（原国家总局令 第28号）**

- [ ] **A2-1. 设备运行状态**
  - 采样探头是否堵塞/破损（拍照固定）
  - 伴热管线是否全程伴热（温度≥120℃），有无打折/泄漏
  - 分析仪量程设置是否合理（是否覆盖实际排放浓度）
  - 数采仪与上位机数据是否一致（打印当场数据与数采仪比对）

- [ ] **A2-2. 标定记录核查（关键）**
  - 调取2026年1月-7月全部标定记录
  - **标气是否在有效期内（核查标气钢瓶标签）**
  - 标定频次是否符合规范（HJ 355要求：SO₂每24h自动标定/每周全量程标定）
  - 比对监测报告是否与自动监测同期开展（手工 vs 在线偏差是否超过±15%）

- [ ] **A2-3. 数据传输完整性**
  - 核查数采仪日志，查看是否存在数据掉线/补录/异常断连记录
  - 超标数据是否如实上传（还是被"工况标记"拦截后未上传）

#### A3. 超标报告义务核查

- [ ] 企业是否在超标发生后1小时内向生态环境部门报告
- [ ] 18次超标对应的《自动监测异常情况报告》是否齐全
- [ ] 报告内容是否与DCS曲线、工况标记记录相互印证

### 🔴 B区：排污许可工艺合规性专项（线索2）

#### B1. 许可工艺与实际工艺一致性核查

**法规依据：《排污许可管理条例》第17条；HJ 847.1（排污许可证申请与核发技术规范）**

- [ ] **B1-1. 许可证核心信息摘录**

  |------|-----------|---------|
  | 治理工艺 | 填写：______ | 现场：______ |
  | 设计处理能力 | 填写：______ Nm³/h | 现场核实设备铭牌 |
  | 许可SO₂排放浓度 | 填写：______ mg/m³ | — |
  | 许可SO₂排放量 | 填写：______ t/a | — |

- [ ] **B1-2. 环评 vs 许可证 vs 实际工艺"三对照"**
  ```
  环评批复工艺：__________
  排污许可证工艺：__________
  现场实际工艺：__________

  判断：
  □ 三者一致 → 手续合规，继续B2核查
  □ 环评 ≠ 许可证 → 许可下达程序瑕疵（需进一步判断是否"批非所建"）
  □ 许可证 ≠ 实际 → 未按许可证生产（许可条例第17条）
  □ 环评 ≠ 实际 → "批建不符"（未验先投/批小建大）
  ```

#### B2. 双碱法脱硫设施运行状态核查（最关键）

**双碱法工艺适配性判断标准：**

| 回转窑废气SO₂浓度区间 | 适用脱硫工艺 | 双碱法适配性 |
|-------------------|------------|------------|
| < 1000 mg/m³ | 单碱法/双碱法（可行） | ✅ 可用，但需精准运维 |
| 1000-3000 mg/m³ | 双碱法/湿法（勉强） | ⚠️ 双碱法效率不稳定，需高运维水平 |
| > 3000 mg/m³ | 湿法（石灰石-石膏） | 🔴 双碱法**难以稳定达标** |

- [ ] **B2-1. 脱硫塔入口SO₂浓度测定**
  - 使用便携式烟气分析仪现场测量脱硫塔入口SO₂浓度
  - 同时读取在线监测数据（出口浓度），计算**实测脱硫效率**
  - 公式：脱硫效率 =（入口-出口）/入口 × 100%
  - 对照双碱法设计效率（通常60-80%），判断是否达标

- [ ] **B2-2. 脱硫浆液pH值实测**
  - 在脱硫塔浆液池或循环泵出口取样
  - 用便携式pH计**现场实测**（要求读数稳定后记录）
  - **记录值：______ pH**（正常范围：8-9）
  - 若pH < 8 → 脱硫效率下降 → 药剂不足或循环异常

- [ ] **B2-3. 脱硫剂库存与投加记录**
  - 现场查看NaOH/石灰石料仓存量（拍照）
  - 调取2026年1-7月投加记录，与超标时段逐一交叉比对
  - **异常发现：超标时段投加量偏低/药剂断供 → 设施不正常运行**

- [ ] **B2-4. 循环泵运行状态**
  - 查看循环泵铭牌电流值
  - 用钳形电流表实测运行电流（记录：______ A）
  - 调取循环泵启停记录，与DCS工况标记时间段交叉比对
  - **若循环泵停运时段与超标时段高度吻合 → 设施不正常运行**

- [ ] **B2-5. 脱硫塔内部状况**
  - 查看塔壁腐蚀/结垢情况（拍照）
  - 查看喷嘴堵塞情况
  - 查看除雾器运行状态（压差是否正常）
  - **塔壁结垢/喷嘴堵塞 → 脱硫效率持续下降**

#### B3. 排污许可证执行情况核查

- [ ] 企业是否按许可证要求的频次开展手工监测（SO₂/颗粒物/NOₓ）
- [ ] 2026年执行报告是否如实报告超标情况
- [ ] 执行报告中脱硫设施运行时间是否与DCS记录一致
- [ ] 许可证要求的上报内容（频次/内容）是否全部履行

## 五、违法行为判定速查

| 发现的异常 | 涉嫌违法 | 现行有效法律依据 |
|-----------|---------|---------------|
| 双碱法设施不正常运行（pH低/药剂不足/循环泵停）导致超标 | 超标排放 + 不正常运行防治污染设施 | 《大气污染防治法》第99条（超标排放）+ 第108条（设施不正常运行） |
| 工况标记时段与DCS实际不符（虚标） | 监测数据弄虚作假（不正常运行防治设施的特殊形态） | 《大气污染防治法》第108条（篡改监测数据）；《污染源自动监控管理办法》第18条 |
| 工况标记100%覆盖所有超标，正常工况数据刻意不标记 | 故意遗漏监测数据 | 同上 |
| 许可证工艺与实际工艺不符 | 未按排污许可证规定运行污染防治设施 | 《排污许可管理条例》第17条 |
| 在线监测设备未按频次校准 | 污染源自动监控设施运行不规范 | HJ 355规范（技术规范，非处罚直接依据）；《污染源自动监控管理办法》 |
| 超标后未按1小时报告 | 未履行报告义务 | 《大气污染防治法》第24条 |
| 许可证下达时工艺本身不达标 | 排污许可证核发程序问题（不属于企业处罚，但需向许可核发机关反馈） | 《排污许可管理条例》第16条 |
