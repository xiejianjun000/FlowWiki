# ZK-2026-07-18-007

> 法律适用正确性审查在2026-08-15前后有本质变化——8-15后新立案禁引废止10法，必须采用法典条款

## 关键论点
- 8-15前评查：对照原单行法条文核实法律适用是否正确
- 8-15后评查：检查是否引用了已被废止的法律（一票否决项）
- 法典新增条款：如生态环境法典第1054条追责期限、第1086/1088条数据印证规则
- 衔接过渡期：仍处于审理中的旧案可引用旧法，8-15后新立案必须引用法典
- 变化对比：法典罚则与旧法罚则有显著差异，需交叉核对
