# ZK-2026-07-18-010

> 9种现场异常信号识别是督察实战的核心技能——涵盖设施运行、台账记录、在线监测、采样点位、危废贮存、排污口、监测数据、环评验收、反向证据九大类

## 关键论点
- 设施运行异常：治理设施未运行/旁路偷排/加药记录与用电量不匹配
- 台账记录异常：运行记录与DCS数据不一致/跳号/缺失关键时段
- 在线监测异常：数据恒值/突变无工况对应/标记停炉启炉频繁
- 危废贮存异常：台账与实物不符/标签缺失/贮存超期
- 反向证据：企业周边水体/土壤异常、居民投诉集中指向
- 每类异常均有对应的证据固定方法（拍照/录像/采样/询问）
