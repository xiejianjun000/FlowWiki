# ZK-2026-07-18-005

> 单一证据不能定案——各类证据须相互印证形成完整、充分的证据链条，这是证据审查的核心原则

## 关键论点
- 证据分类：主体证据/违法事实证据/定性证据/危害后果证据/裁量证据
- 单一依赖红线：仅凭当事人陈述或询问笔录作为唯一处罚证据即触发一票否决
- 印证关系：书证+物证+视听资料须在时间/地点/数量上相互吻合
- 现场笔录+监测报告+询问笔录是数据造假类案件的"铁三角"证据链
- 数据异常信号：监测数据与企业工况标记不一致时，须做水平衡/物料平衡核算
