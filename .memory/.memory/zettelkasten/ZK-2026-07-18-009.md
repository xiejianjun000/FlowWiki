# ZK-2026-07-18-009

> 一般行政处罚案卷评查是六类案卷中最通用的核心评查流程，覆盖立案→检查采样→询问→告知五要素→法制审核→集体讨论→决定→送达全链条

## 关键论点
- 通用性最强：几乎所有行政处罚案件均可套用此流程框架
- 每个步骤需调用对应的单项SOP：证据链核验→采样合法性→程序违法排查→法律适用审查→送达合规
- 六类案卷评查共性：25项一票否决清单是前置预筛，所有案件必须先扫
- 全流程步骤已预置一手提示词，可一键生成评查报告
- 包含该类案件专项项（如一般处罚的告知五要素完整性）
