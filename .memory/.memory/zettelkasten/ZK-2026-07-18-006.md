# ZK-2026-07-18-006

> 采样程序合法性审查是执法案卷败诉的首要原因——点位布置、空白样、样品保存、运输链任一环节出问题即证据不可采

## 关键论点
- 监测数据进入诉讼后，被告律师第一攻击点永远是采样程序合法性
- 关键审查点：采样点位是否按规范布置、现场空白样和平行样是否齐全、样品保存和运输是否符合标准、采样记录是否完整
- 采样不合法→监测报告不可采→违法事实证据链断裂→可能触发一票否决
- 评查时若含监测数据，必须先过采样关再谈证据印证
- 法典第1086/1088条确立了数据印证规则，强化了采样合法性的前置地位
