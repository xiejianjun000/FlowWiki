# ZK-2026-07-18-004

> 程序违法排查是评查中最常见的红线命中来源——告知权、期限、法制审核、集体讨论、回避、时序六大类

## 关键论点
- 告知五要素：事实（何人/何时/何地/何种行为/何种后果）+ 陈述申辩权 + 听证权 + 复议权 + 诉讼权
- 期限铁律：陈述申辩期≥3日、听证期≥5日、听证通知提前7日——注意工作日vs自然日区别
- 法制审核：重大复杂案件未经法制审核即决定属一票否决项
- 集体讨论：重大复杂案件处罚决定须经负责人集体讨论，缺少即程序违法
- 时序红线：常见"先调查后立案""先签发后告知""陈述申辩期未满即决定"均属程序违法
