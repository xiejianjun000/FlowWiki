# ZK-2026-07-18-008

> 送达合规审查是程序合法性最后一道关——方式选择错误、回证缺失、公告送达条件未满足均属常见问题

## 关键论点
- 送达方式等级：直接送达（首选）→留置送达→邮寄送达→公告送达（最后手段）
- 公告送达必须满足"其他方式均无法送达"的前提条件，且经法制审核确认
- 回证缺项：送达日期、送达人签名、受送达人签名/拒收注明、见证人签名——缺一不可
- 期限预警：处罚决定书须在法定期限内送达，逾期送达影响执行效力
- 直接送达至当事人住所/办公场所视为有效，交同住成年家属签收须注明关系
