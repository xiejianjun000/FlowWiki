# ZK-2026-07-18-002

> 评查标准体系是执法案卷评查的"宪法"，分合法性审查（零分项）和规范性审查（扣分项）两大部分

## 关键论点
- 合法性审查=25项一票否决，规范性审查=100分扣分制
- 合法性项全部通过后，再进入规范性打分
- 2026-08-15法典生效后，法律适用类审查标准有重大调整（禁引废止10法）
- 评查程序：接收案卷→预筛→合法性审查→规范性审查→集体讨论→报告→归档
