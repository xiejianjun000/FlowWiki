# ZK-2026-07-18-001

> 25项一票否决清单是评查合法性审查的红线——任一命中即案卷整体不合格（零分），优先于规范性打分

## 关键论点
- 25项覆盖主体/管辖、事实证据、法律适用、程序、文书送达五大类
- 任一命中即触发一票否决，不进入规范性评分
- 核验流程：先扫25项 → 命中则标记违反项+证据位置 → 再进规范性打分（不重复扣分）
- 涉及法典8-15后新标准的项目有专项标注
