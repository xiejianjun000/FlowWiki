## 何时读我（触发信号）
- 审查完一份案卷，要交付"评查报告"，不知道标准结构怎么排。
- 需要写"⚖️ 合规守门报告"或"评查记分表"，要求：合法性结论 + 规范性得分 + 改进建议。
- 改进建议不知道怎么引"具体评查细则序号 + 法条"，怕写空话。

## 目标拆解
**本篇要解决的问题**：把零散审查发现，组织成结构清晰、依据充分、可直接上报的评查报告。
**最终产出** = 一份标准 ⚖️ 合规守门报告（含评查记分表），每条问题都"有细则序号 + 有法条 + 有证据位置"。

> 核心心法：**三块结构固定**（合法性结论→规范性得分→改进建议）；**建议不写空话**，必须引到"评查细则第X项 + 法条第X条"两个锚点。

## 资料准备清单
- [ ] 已完成的审查记录（合法性预筛 + 规范性逐项评分）
- [ ] [[25项一票否决清单]]（用于合法性结论引用）

## 全流程步骤（逐步可跟）
1. **填案卷基本信息** → 案件名称、案号、当事人、处罚内容、结案日期、案卷类型。
2. **写【一、合法性结论】** → 对照 [[25项一票否决清单]]：
   - 无命中 → "经核查，未发现一票否决项，合法性审查合格。"
   - 有命中 → 逐条列：第X项 + 违反《评查细则2024》原文 + 证据位置（Pxx）+ 法条。
   - 涉法律适用：若结案 ≥ 2026-08-15 仍引旧 10 法，标红引用 [[法典8-15施行倒计时看板]] 与法典§1242。
3. **写【二、规范性得分】** → 套用评查细则公式：
   - 卷面基本要素扣分（20分制）
   - 各类型文书得分（100分制，按 [[评查细则全文2024指针]] 各项）
   - 总分 = 50 + 50×(文书实际得分和 / 标准分和 − 基本要素扣分值)
   - 注明：存在重大问题则整体 0 分。
4. **写【三、改进建议】** → 每条格式："问题现象 + 依据评查细则2024 第X项 + 《XX法》第X条 + 整改建议"。禁止"建议加强学习"类空话。
5. **出评查记分表（附表）** → 表格化逐项错误，列：评查项目/序号/扣分或重大问题/依据/证据位置。
6. **定稿复核** → 法条临界再查 [[法典8-15施行倒计时看板]]；建议引用的细则序号与法条是否准确到条款项。

## 常见扣分点 / 易错点（报告写作 Top 7）
- **合法性结论含糊**：只写"基本合法"，不列具体 25 项核查结果，上报被退回。
- **建议写空话**："请规范执法""加强管理"无细则序号、无法条，等于没写。
- **法条不引到条款项**：写"违反大气污染防治法"而非"第X条第X款"，定性不成立。
- **8-15 红线漏标**：旧法引用未在报告中标注临界风险，后续被上级一票否决。
- **得分公式用错**：漏减基本要素扣分，或把"重大问题"也计入得分（应直接 0 分）。
- **证据位置缺失**：每条问题不标 Pxx，复核与整改无从查证。
- **复议/诉讼期限告知错未提**：应告"6个月起诉（行政诉讼法§46）""60日复议（行政复议法§20）"，写错属规范性扣分，报告中须点出。
