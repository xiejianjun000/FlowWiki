## 何时读我（触发信号）
- 上级/法制部门甩给你一份案卷（一般处罚、不予处罚、按日计罚、查封扣押、移送拘留/犯罪等），让你"评一下"。
- 你第一次参与案卷评查，不清楚"先做什么、后做什么、最后交什么"。
- 不确定这份案卷会不会被撤销/败诉，需先判断红线。

## 目标拆解
**本篇要解决的问题**：零基础上手，把"收到案卷"到"出评查结论"走通一遍。
**最终产出** = ⚖️ 一份合规守门报告（含：合法性预筛结论 + 规范性得分 + 改进建议），可直接归档/上报。

> 核心心法：**先合法性（一票否决），后规范性（扣分/打分）**；合法性命中即整体 0 分，规范性打分才有意义。

## 资料准备清单
- [ ] 案卷全套材料（按时间顺序排列：立案→调查→告知→决定→送达→执行→结案）
- [ ] 案卷类型判定：属 6 类中哪一类（一般处罚 / 不予处罚 / 按日连续处罚 / 查封扣押 / 移送拘留 / 移送涉嫌犯罪）
- [ ] [[25项一票否决清单]]（第一步先看它）
- [ ] [[评查细则全文2024指针]]（对照每条评分项）
- [ ] [[法典8-15施行倒计时看板]]（看结案日期是否跨 8-15 临界点）
- [ ] 对应的专项 SOP：[[采样程序合法性审查]]、[[证据链闭环核验]]、[[法律适用正确性审查]]、[[程序违法排查]]、[[送达合规审查]]

## 全流程步骤（逐步可跟）
1. **接收与分类** → 确认案卷类型，调对应 #实战全流程 文档（如 [[一般行政处罚案卷评查]]）了解该类特有审查点。
2. **合法性预筛（第一步必做）** → 调 [[25项一票否决清单]]，逐条扫 25 项（主体/管辖、违法主体、事实证据、法律适用、程序、送达执行等）。命中任一即标记"重大问题"。
3. **法律适用临界核查** → 查结案日期：若 **≥ 2026-08-15**，必须适用生态环境法典，引用已废止 10 法 = 一票否决。详见 [[法典8-15施行倒计时看板]]，对照 [[法典对照与废止10法]]。
4. **逐项实质审查（按 SOP 调子文档）**
   - 证据/采样 → [[采样程序合法性审查]] + [[证据链闭环核验]]（败诉首要原因：点位/空白/保存/资质）
   - 法律适用 → [[法律适用正确性审查]]（法条须引用到条、款、项）
   - 程序 → [[程序违法排查]]（告知五要素、期限、法制审核、集体讨论）
   - 送达 → [[送达合规审查]]（方式/回证/期限）
5. **规范性打分** → 调 [[评查细则全文2024指针]]，按"卷面基本要素（20分）+各类型文书（100分）"逐项评分。
   - **不重复扣分原则**：同一瑕疵已在合法性环节标重大问题、或在某一要素扣过，不跨项重复扣。
   - 公式：案卷得分 = 合法性得分(50) + 规范性得分(50 × 实际得分和 / 标准分和 − 基本要素扣分值)。注意：存在任一项重大问题直接 0 分。
6. **出 ⚖️ 报告** → 套用下方"输出成果模板"，写合法性结论 + 规范性得分 + 改进建议（建议须引具体评查细则序号 + 法条）。

## 常见扣分点 / 易错点（Top 8）
- **先打分后扫红线**：顺序颠倒，漏掉一票否决项，整卷 0 分风险。
- **只看法条名不看条款项**："大气污染防治法"式引用，缺条/款/项 = 合法性重大问题（清单第10项）。
- **8-15 临界点漏查**：结案跨 8-15 仍引旧 10 法，一票否决（[[法典8-15施行倒计时看板]]）。
- **重复扣分**：同一问题在多个评分项目里各扣一次，违反"不重复扣分"。
- **时间逻辑混乱**：先调查后立案、先签发后告知，属程序重大问题（清单第21、14项）。
- **告知期限按自然日算**：陈述申辩 3 日、听证 5 日均为工作日，按自然日算 = 程序违法。
- **起诉/复议期限告知错**：应告"6个月起诉""60日复议"，写成 3 个月/15 日 = 规范性扣分。
- **签名细节缺失**：笔录末页无人签名、无"以下空白"、涂改无压指印 = 基本要素扣分。

```

【一、合法性预筛结论】对照 [[25项一票否决清单]] 25项：
  ✓ 未发现重大问题 / ✗ 发现 N 项重大问题：
    · 第X项：违反《评查细则2024》"……" + 证据位置（Pxx）
    · 第X项（法律适用）：引用已废止《XX法》（法典§1242，8-15后禁引，见看板）
  ⟶ 结论：合格 / 整体0分（任一命中）
  卷面基本要素扣分值：___ / 20
  各类型文书得分：___ / 100（按 [[评查细则全文2024指针]] 逐项）
  总分 = 50 + 50×(文书实际/标准 − 基本要素扣分) = ___
【三、改进建议】（每条引细则序号+法条）
  · 建议1：……（依据评查细则2024 第X项 + 《XX法》第X条）
  · 建议2：……
```
