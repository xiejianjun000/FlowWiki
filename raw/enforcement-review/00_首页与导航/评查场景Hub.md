# 评查场景Hub

> 案卷评查任务的统一入口。选案件类型 → 进入实战流程 → 调SOP+法条指针 → 一键评查。

## 使用方式

1. **确定案件类型** → 从下方6类中选一个
2. **先扫25项红线** → [[25项一票否决清单]]（所有类型必扫）
3. **进入实战流程** → 按TRAE六步法逐步执行
4. **查法条** → 调用法条速查指针
5. **填报告** → 使用[[评查报告模板（可直接使用）]]
