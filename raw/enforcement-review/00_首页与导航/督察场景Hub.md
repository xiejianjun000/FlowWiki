# 督察场景Hub

> 环保督察任务的统一入口。选任务类型 → 进入流程 → 加载方法 → 输出报告。

## 使用方式

1. **确定督察阶段** → 进驻准备/现场核查/问题反馈/整改跟踪
2. **进入流程SOP** → 按六步法逐项执行
3. **加载现场方法** → 异常信号识别+博弈话术
4. **输出正式文书** → 反馈意见+专报+整改台账
