- 权威依据已入仓：本库 查封扣押办法

## 目标拆解
**问题**：查封扣押是行政强制措施，对象须与违法直接相关、程序须审批/当场告知、期限须遵守、逾期须解除或延期批准，任一错即重大问题。
**产出**：对象适当性 + 程序合法性 + 专项红线核查 + ⚖️ 合规守门报告。

> 第一步仍调 [[25项一票否决清单]]；专项否决项集中在评查细则2024§一 项9、13、16、25。

## 过程检查（逐步核验 · TRAE 第三步）


### 第0步 · 合法性预筛
→ [[25项一票否决清单]] 评查细则2024§一 序号1-25。主体/证据/程序通用红线仍致命。

### 第1步 · 对象适当性审查
- 🔴 红线（项13）：**查封、扣押与违法行为无关的设施、设备、物品、工具或场所** = 重大问题（不同法律对查封对象有不同规定，须对应）。
- 🔴 红线（项9）：对未处于生产状态的建设项目可能产生污染排放的行为实施查封（兜底条款无依据，项11同类）。
- ⚠️ 对象须限于"造成或可能造成严重污染"的设施/物品，不得超范围。

### 第2步 · 审批与实施程序
→ [[程序违法排查]]
- 🔴 红线（项16）：查封、扣押**未当场告知**理由、依据、权利救济途径，未听取陈述申辩 = 重大问题。
  - 注意事项：当场实施的，可在现场检查（勘察）笔录中告知；但**未制作查封、扣押现场笔录** = 重大问题。
- 一般须**经本级生态环境主管部门负责人批准**后方可实施（审批表）；情况紧急当场实施的，须在24小时内补办审批。
- 🔴 红线（项1）：无管辖权/以分局或机构名义实施/落款与公章不符。

### 第3步 · 期限与解除（🔴 高频致命）
- 🔴 红线（项25）：**查封、扣押超期未作出解除决定，或者未经本级生态环境主管部门负责人批准延长查封、扣押期限** = 重大问题。
- 法定期限：一般不得超过30日；情况复杂的，经负责人批准可延长，但须书面通知当事人。
- 经查不再需要查封扣押的，应**及时解除**并制作解除决定书。

### 第4步 · 证据与清单
→ [[证据链闭环核验]]
- 查封扣押清单须列明物品名称、数量、规格，由当事人、见证人、执法人员签名。
- 证明"造成或可能造成严重污染"的证据（监测报告/现场情况）须入卷。

### 第5步 · 法律适用审查（含8-15法典临界）
→ [[法律适用正确性审查]] + [[法典8-15施行倒计时看板]]
- 🔴 红线（项8-10 / 第10项）：依据须准确到条/款/项；结案≥2026-08-15 引已废止查封扣押办法（部门规章，注意配套清理）或旧10法 = 一票否决。

### 第6步 · 文书与送达
→ [[送达合规审查]]
- 查封扣押决定书、现场笔录、解除决定书均须依法送达/签收。

## 结果优化（复核与交叉互评 · TRAE 第四步）
> 打完分后做三件事：(1) 用 [[25项一票否决清单]] 再核一遍红线；(2) 交叉互评——差异超阈值时复核（参考知识库“一案双评”经验）；(3) 裁量/法律适用是否过罚相当（呼应复议条例适当性审查）。

## 常见扣分点 / 易错点（Top 8）
1. 超期未解除且未批延期（项25 零分）
2. 查封与违法无关设施（项13 零分）
3. 未当场告知/无现场笔录（项16 零分）
4. 以分局/机构名义作出决定（项1）
5. 兜底条款无依据实施查封（项9/11）
6. 清单无当事人/见证人签名
7. 延期未书面通知当事人
8. 8-15临界法律适用漏改

## 该类案件专项一票否决项（区别于通用25项）
查封扣押**专项致命项（评查细则2024§一项9、13、16、25）**，叠加通用25项：
- **项9（关联）**：对未处于生产状态的建设项目可能排放行为实施查封（兜底条款无依据）。
- **项13**：查封、扣押与违法行为无关的设施、设备、物品、工具或场所。
- **项16**：未当场告知理由/依据/救济途径、未听取陈述申辩，或未制作查封扣押现场笔录。
- **项25**：查封、扣押超期未解除，或未经负责人批准延长。
- ⚠️ 任一项命中 = 整体零分，核心在"对象适当 + 当场告知 + 期限解除"。
