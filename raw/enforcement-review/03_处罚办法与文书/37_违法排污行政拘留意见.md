# 违法排污行政拘留 — 法条精确化

> 依据：生态环境法典 第1179条 | 原文件：行政主管部门移送适用行政拘留环境违法案件暂行办法

## 第1179条 行政拘留适用情形

| 情形 | 处罚 |
|------|------|
| 擅自倾倒/堆放/丢弃/抛撒/遗撒/焚烧固体废物，造成严重后果 | 直接责任人：10-15日拘留；情节较轻→5-10日 |
| 未采取防范措施，造成危险废物扬散/流失/渗漏或其他严重后果 | 同上 |
| 未经批准擅自转移危险废物 | 同上 |
| 在生态保护红线/永久基本农田/特别保护区域内建设固废/危废处置设施或垃圾填埋场 | 同上 |
| 无许可证或未按许可证从事收集/贮存/利用/处置危险废物 | 同上 |
| 将危险废物提供或委托给无许可证单位或个人 | 同上 |

## 移送公安衔接
- 第1058条：行政拘留可折抵刑期；行政罚款可折抵罚金
- 暗管/渗井/渗坑排放 + 篡改/伪造监测数据 → 移送公安，罚款10万-100万元
- 情节严重：吊销许可证，停业、关闭

## 关联法条
- 第1179条：行政拘留适用情形及处罚标准
- 第1058条：民行刑衔接条款
- 第1060条：按日连续处罚
- 第1065条：消除污染/修复生态责任
