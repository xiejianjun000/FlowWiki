# 五、评查实务要点

## 5.1 评查流程

```
1. 形式审查 → 检查案卷完整性、装订规范
2. 零分项筛查 → 逐项对照合法性评查标准
3. 规范性评分 → 逐项对照扣分标准
4. 汇总得分 → 计算总分，确定等级
5. 问题反馈 → 列出问题清单及依据
6. 建议改进 → 提出整改建议
```

## 5.2 零分项筛查优先级

必须**最先排查**的致命问题（按重要性排序）：

1. **执法主体**：处罚主体是否有法定权限？
2. **程序告知**：是否履行了事先告知、听证告知？
3. **证据充分**：是否有足够证据支持处罚决定？
4. **法律适用**：法律是否正确、有效、引用到条款项？
5. **移送义务**：涉嫌犯罪是否移送？
6. **处罚程序**：是否存在先调查后立案、先签发后告知？

## 5.3 规范性评查重点

### 立案阶段
- 立案审批表填写完整、及时
- 立案时间应在调查取证之前

### 调查取证阶段
- 两名以上执法人员（同一人不得同时出现在不同时间不同地点）
- 出示执法证件（应在笔录中体现）
- 证据链完整（多种证据相互印证）
- 笔录规范（签名、日期、涂改确认、空白处理）

### 告知阶段
- 事先告知书内容完整（事实五要素+法律依据+处罚内容）
- 听证告知（较大数额罚款等）
- 陈述申辩权告知
- 期限计算正确

### 决定阶段
- 处罚决定书内容完整
- 集体讨论记录（重大案件）
- 法律引用准确到条款项
- 救济途径告知正确
- 盖章正确、日期准确

### 送达执行阶段
- 送达方式合法
- 送达回证完整
- 罚款缴入财政专户（代收银行收据）
- 结案材料齐全

### 归档阶段
- 三孔一线装订
- A4纸张统一
- 去除金属物
- 页码、备考表齐全
- 材料按时间顺序排列

## 5.4 常见评查误区

1. **只看结果不看过程**：不能只看处罚决定书，要审查全流程
2. **忽视时间逻辑**：立案→调查→告知→决定→送达，时间顺序必须正确
3. **忽略签名细节**：签名是文书效力的关键，缺签=文书无效
4. **轻视程序**：程序违法往往比实体问题更致命
5. **法律引用粗疏**：只看法律名称不看具体条款
6. **忽视救济告知**：诉权告知错误可能导致处罚决定被撤销
7. **证据关联性忽视**：证据虽然存在但与违法事实无关联

## 5.5 评查报告写作要点

一份规范的评查报告应包含：
1. **案卷基本信息**：案件名称、案号、当事人、处罚内容
2. **评查依据**：所依据的评查标准
3. **合法性评查结果**：逐项说明是否触及零分项
4. **规范性评查结果**：逐项列出扣分项及扣分值
5. **总分及等级**：百分制得分、对应等级
6. **问题清单**：按严重程度排序
7. **改进建议**：针对性整改建议
8. **评查人签名**：评查人及日期
