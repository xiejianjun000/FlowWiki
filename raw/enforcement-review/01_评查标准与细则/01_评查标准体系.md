# 一、评查标准体系

## 1.1 核心标准文件
- **《生态环境行政执法案卷评查细则（2024年修订版）》**——生态环境部制定，全国统一标准
- 各省可能在此基础上制定更细化的补充标准（如安徽省有省级标准）

## 1.2 评查总分：100分

### 合法性评查（零分项——触及即整卷0分）
触及以下任何一项，案卷总分直接计0分：

**（一）执法主体资格**
1. 实施行政处罚的部门没有法定处罚权限的
2. 实施行政处罚的部门没有获得法定授权权限的
3. 行政处罚决定书、责令改正违法行为决定书没有加盖行政机关印章或者加盖印章模糊不清、无法辨认的
4. 行政处罚决定书、责令改正违法行为决定书加盖的印章与作出处罚决定的行政机关不一致的
5. 执法人员不具备执法资格的
6. 环境保护主管部门对涉嫌犯罪案件没有依法移送司法机关处理的
7. 行政处罚决定没有证据或没有足够证据支持行政处罚内容的

**（二）违法主体认定**
1. 被认定为违法主体的当事人不具有违法行为的实施能力的
2. 违法主体认定不当，行政处罚决定中的违法主体认定与实际违法主体不一致的

**（三）违法事实证据**
1. 没有足够的违法事实，不能认定为违法行为成立的
2. 确定违法事实所使用的证据不合法的
3. 证据要求缺失，无法确定违法事实或确定违法事实不清的

**（四）法律适用**
1. 确定违法事实所适用的法律、法规、规章已废止、无效或尚未生效的
2. 确定违法事实所适用的法律、法规、规章不正确，没有引用到条、款、项、目的
3. 行政处罚决定超越法定权限和范围的
4. 行政处罚决定超出法定期限，并且无法补正的
5. 行政处罚决定超出处罚权限标准或超过自由裁量权标准规定，且无正当理由的

**（五）执法程序（严重程序违法——重大违法行为）**
1. 未履行事先告知的（形成重大程序违法）
2. 在作出责令停产停业、吊销许可证或者较大数额罚款等行政处罚决定前，未告知当事人有要求举行听证的权利的
3. 作出处罚决定前没有依法告知当事人的违法事实、理由和依据的，或作出的处罚决定与事实不符的
4. 没有履行行政处罚事先告知程序的
5. 没有告知当事人陈述、申辩的权利的
6. 没有告知当事人依法享有申请行政复议或者提起行政诉讼权利的
7. 没有告知当事人救济途径的
8. 未依法送达行政处罚决定书、责令改正违法行为决定书的
9. 应当依法移送的案件未移送的
10. 应当依法举行听证而未举行的
11. 未经集体讨论作出处罚决定的（重大案件）

## 1.3 规范性评查（一般标准——逐项扣分）
一般标准指违反本细则规定的一般标准中的操作规范，可能出现但不致造成行政处罚决定被撤销、确认违法或变更撤销的问题。扣分值设定为10分、5分、3分、2分、1分。

### 逐项评分标准（部分关键文书）

**立案审批表**
- 没有填写立案审批表或者补充立案审批表不符合填写制作规定的 → 扣5分
- 立案审批表主要事项填写不全、不准确的 → 扣2分
- 存在涂抹、空白处未划线处理的 → 扣1分

**现场检查（勘察）笔录**
- 同一执法人员同一时间在不同地点检查 → 扣5分
- 一份笔录同时记录被检查当事人、被询问当事人 → 扣5分
- 笔录末页被检查当事人、在场人没有签字确认或者没有注明"以上笔录我看过，与我所说一致"，执法人员没有注明情况的 → 扣3分
- 两名执法人员没有签字或者只有一人签字 → 扣3分
- 现场（检查）的关键要素没有记录（当事人名称、违法行为、产品等）可作处可不作处涂改未处 → 扣2分
- 没有记载时间、地点或经过情形准确的 → 扣2分
- 被检查当事人或在场人没有逐页签字 → 扣2分
- 现场照片没有注明时间、地点、制作人信息（每张）→ 扣3分

**调查询问笔录**
- 同一执法人员同一时间询问不同被询问人 → 扣5分
- 一份记录同时询问不同被询问人 → 扣5分
- 没有告知被询问人享有权利及法律依据 → 扣2分
- 记录必须内容（当事人名称、地址、行为等），可作处未盖章、加盖印章未加盖 → 扣5分

**先行登记保存证据**
- 未告知当事人先行登记保存的时限、地点的 → 扣5分
- 先行登记保存物品清单没有当事人签名或盖章的 → 扣5分
- 没有准确的时间、地点描述 → 扣3分

**行政处罚事先告知书**
- 没有制作行政处罚事先告知书的 → 扣5分
- 没有准确写明处罚机关名称，加盖印章、或者未加盖 → 扣5分
- 没有准确写明处罚的法律依据、拟处罚内容的 → 扣5分

**行政处罚决定书**
- 没有行政处罚决定书的 → 扣5分
- 没有准确写明当事人名称或地址的 → 扣3分
- 没有准确写明处罚机关名称，加盖印章、或者未加盖 → 扣5分
- 违法行为事实不清楚或者证据不充分的 → 扣3分
- 没有准确写明处罚的法律依据、处罚内容的 → 扣5分
- 没有当事人签名或盖章的 → 扣3分
- 没有准确写明作出处罚决定的时间 → 扣3分
- 没有准确写明救济途径的 → 扣3分
- 没有收到处罚决定书的时间 → 扣3分

**案件调查终结报告**
- 没有准确写明案件来源、当事人情况 → 扣3分
- 没有准确写明违法事实、情节的 → 扣3分
- 没有准确写明证据及分析意见 → 扣3分
- 没有准确写明处罚意见 → 扣3分
- 没有处罚处理意见、证据不充分的 → 扣3分
- 没有其他事项 → 扣3分
- 涂改空白处未处理 → 扣2-3分

**案件讨论记录**
- 案件讨论未通过集体讨论的 → 扣5分
- 当事人陈述申辩后，没有集体讨论记录的 → 扣5分
- 讨论成员签名不齐全 → 扣5分
- 没有关键讨论时间 → 扣3分
- 三名成员签名齐全但缺少 → 扣2分
- 没有主要领导签字 → 扣3分
- 主要领导没有签字的讨论 → 扣5分
- 没有地方计划（方案）→ 扣5分
- 主要领导签字没有的 → 扣5分

## 1.4 评查等级划分
- **优秀**：90分以上
- **良好**：80-89分
- **合格**：60-79分
- **不合格**：59分以下（零分卷直接不合格）

## 1.5 评查方式
- 一案一评
- 一卷一表
- 交叉互评、一案双评
- 初评 + 复评（专家复审）
- 全面评查、随机抽查、重点抽查
