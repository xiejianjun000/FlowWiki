## 何时读我（触发信号）
- 准备做"证据三性"审查（合法性/真实性/关联性）
- 案卷证据仅有一份询问笔录或一份监测报告
- 不同证据之间出现时间/地点/数量矛盾
- 评查进入"案件调查报告"证据链评分项

## 目标拆解
单一证据不能定案；各类证据须**相互印证**形成完整、充分的证据链条。核验目标 = 找出断点、矛盾点、单一依赖点。
最终产出 = ⚖️ 证据链闭环结论（印证关系图 + 断点清单 + 是否触及一票否决）。

## 资料准备清单
- [ ] 现场检查（勘察）笔录
- [ ] 调查询问笔录（当事人/证人）
- [ ] 监测/检测/鉴定报告
- [ ] 书证（营业执照、环评、验收、排污许可证复制件）
- [ ] 视听资料（照片、录像及说明）
- [ ] 水平衡 / 物料平衡核算底稿（必要时）
- [ ] [[采样程序合法性审查]]（先确采样合法，再谈印证）

## 全流程步骤（逐步可跟）
1. **列证据清单** → 按"主体证据/违法事实证据/定性证据/危害后果证据/裁量证据"分类（法典新标准4.3证据目录）。
2. **先过采样关** → 若含监测数据，先调 [[采样程序合法性审查]] 确认采样合法，否则数据不可采。
3. **核单一依赖** → 是否仅有当事人陈述/询问笔录作为唯一处罚证据（评查细则§3重大质量问题"证据材料不足以证明…予以处罚"；实务"单一证据不定案"）。
4. **交叉印证** → 书证+物证+视听资料是否互相支持：笔录记载的违法持续时间/数量 ↔ 监测报告 ↔ 照片录像 ↔ 台账。
5. **查矛盾点** → 证据之间指向的违法事实互相矛盾且无合理解释（评查细则§3常见错误"证据互相矛盾"）。
6. **核复制件** → 书证复制件须注明"经核对与原件无误"、出处、调取时间、提供人+执法人员签章（评查细则§5扣2分；02_常见错误 2.5(3)）。
7. **核视听说明** → 照片/录像须注明证明对象、拍摄时间、地点、拍摄人、证明内容（评查细则§5扣1分）。
8. **异常信号扫描** → 水平衡/物料平衡：进水≫排水、原料投入≫产物+库存、用电量/药剂量与处理规模不匹配 → 偷排/暗管嫌疑（实务经验在线监测异常识别）。
9. 命中问题 → 引用 [[25项一票否决清单]] 第7/8项 或 [[评查细则全文2024指针]]。

## 常见扣分点 / 易错点（Top 8）
- **🔴 单一证据定案**：把询问笔录/当事人陈述作为唯一处罚证据（评查细则§3重大质量问题；02_常见错误2.5(2)）。
- **🔴 证据互相矛盾无合理解释**：笔录与监测报告/数量不一致（评查细则§3"证据指向违法事实互相矛盾"）。
- **🔴 主要证据缺失**：需采样案件无采样合法性证明（评查细则§3）。
- **🟠 复制件未核对签章**：未注"经核对与原件无误"+提供人/执法人员签章（评查细则§5扣2分；02_常见错误2.5(3)）。
- **🟠 照片/录像无说明**：缺时间/地点/拍摄人/证明内容（评查细则§5扣1分）。
- **🟠 当事人拒签无见证人**：未注明原因且无见证人（02_常见错误2.5(4)）。
- **🟡 水平衡/物料平衡异常未核实**：进水与排水、原料与产物显著失衡未追查 → 遗漏偷排线索。
- **🟡 证人证言不规范**：缺证人基本情况/签名/日期（评查细则§5扣1分）。
