## 何时读我（触发信号）
- 案卷含监测/检测/鉴定报告，且处罚事实依赖采样数据
- 涉及一类污染物（重金属、氰化物等）超标认定
- 评查进入"证据三性（合法性/真实性/关联性）"环节
- 出现监测数据异常（超标倍数存疑、点位存疑、空白缺失）

## 目标拆解
采样程序 = 监测数据能否作为定案证据的**前提**。任一核心环节缺失 → 数据不具法律效力 → 事实不清（一票否决）。
最终产出 = ⚖️ 采样程序合规结论（点位/空白/保存/检定/见证 全项核验 + 命中法条）。

## 资料准备清单
- [ ] 现场检查（勘察）笔录（须记载采样情况）
- [ ] 取样记录 / 采样原始记录单
- [ ] 监测/检测/鉴定报告（含采样点位、时间、方法、仪器）
- [ ] 样品保存与运输记录（空白样、冷藏记录、送检交接单）
- [ ] 计量器具检定/校准证书
- [ ] 见证人签字材料（当事人拒签时）
- [ ] 相关技术规范（HJ 91.1 废水、HJ/T 194 废气、HJ/T 166 土壤等）

## 全流程步骤（逐步可跟）
1. **核点位** → 一类污染物必须在**车间排放口**（车间或生产设施废水排放口）采样，绝不可用总排放口数据替代（[[25项一票否决清单]] 第8项关联；法典§1080 评查要点"超标采样须在一类污染物排放口采集"）。
2. **核采样孔位置** → 废气采样孔应避开弯头、阀门、变径管，距弯头下游≥6D、上游≥3D（D为管径），保证混合均匀、代表性（监测技术规范要求；点位须与取样记录一致 → 评查细则监测报告项"点位与取样记录采样点位一致"）。
3. **核空白样** → 须有**全程序空白**（随样品全程运输保存）与**运输空白**（运输过程控制），QA/QC 缺失即真实性存疑（实务经验评查红线）。
4. **核保存与送检** → 样品须按规范**4℃冷藏、24小时内送检**，超期/未冷藏 = 样品变质风险、数据不可采。
5. **核计量仪器** → 采样、分析所用计量器具应在检定/校准有效期内（法典§1086：未检定/超期的设备数据不具备法律效力）。
6. **核见证人** → 当事人拒签采样记录/笔录的，应注明原因并有**见证人签字或盖章**（[[02_常见错误与问题]] 2.5(4)；现场笔录制作规范）。
7. **交叉比对** → 取样记录点位 ↔ 监测报告点位 ↔ 现场检查笔录记载，三者不一致即断链（评查细则监测报告项多家扣分0.5分）。
8. 命中问题 → 引用 [[法典8-15施行倒计时看板]] 或 [[评查细则全文2024指针]] 锁定条款。

## 常见扣分点 / 易错点（Top 8）
- **🔴 一类污染物未在车间排放口采样**：程序违法，实务列为"一票否决"级（评查细则§3："涉及需采样的案件无证据证明采样过程合法性"；法典§1080）。
- **🔴 采样点位与取样记录/监测报告不一致**：评查细则监测报告项"点位与取样记录一致"扣0.5分（一般/不予/按日/查封/移送类均设该考核项）。
- **🟠 全程序空白 / 运输空白缺失**：证据真实性存疑，评查列为高频扣分项（实务经验"证据合法性存疑"）。
- **🟠 超期送检 / 未4℃冷藏**：样品保存不合规，数据效力受质疑（技术规范 + 实务经验）。
- **🟠 计量器具超期未检定**：法典§1086 明确超期未检设备数据不具法律效力。
- **🟠 采样孔位置不合规（距弯头<6D/>上游<3D）**：代表性不足，易被复议/诉讼推翻。
- **🟡 当事人拒签无见证人/未注明原因**：02_常见错误 2.5(4)，笔录证明力受损。
- **🟡 监测报告缺资质标志/方法/仪器**：评查细则监测报告项"资质认定标志+监测字号"缺失本项得0分。
