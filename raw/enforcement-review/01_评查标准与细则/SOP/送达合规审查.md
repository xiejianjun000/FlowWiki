## 何时读我（触发信号）
- 评查送达执行阶段（卷面基本要素"送达回证"扣2分）
- 送达回证缺失/无签收日期/无见证人
- 采用留置/邮寄/公告送达，条件存疑
- 办案超期（立案至决定/送达超过法定期限）

## 目标拆解
送达 = 文书发生法律效力的节点。方式违法/回证缺失 = 程序瑕疵甚至重大违法。
最终产出 = ⚖️ 送达合规结论（方式合规 + 回证完整 + 期限预警）。

## 资料准备清单
- [ ] 各类文书送达回证
- [ ] 行政处罚决定书 + 事先告知书 + 责令改正决定书等
- [ ] 留置送达：见证人签字 / 拍照录像
- [ ] 邮寄送达：邮寄凭证 + 签收记录
- [ ] 公告送达：公告材料 + 期满证明
- [ ] 立案审批表 / 结案审批表（核办案期限）
- [ ] [[程序违法排查]]（告知/决定时序前置）

## 全流程步骤（逐步可跟）
1. **核优先方式** → **直接送达优先**；受送达人本人/同住成年家属/法定代表人/负责收件人签收（02_常见错误2.4送达顺序）。
2. **核回证** → 每份文书须有**送达回证**（卷面基本要素扣2分；02_常见错误2.5(1)告知书须签章注明时间）。受送达人拒签的，注明情况并由**见证人**签字（实务送达审查表）。
3. **核留置送达** → 受送达人拒收的，留置送达须有**见证人见证或拍照/录像**记录，符合法定条件（实务经验"留置送达无见证人=送达瑕疵"；高频扣分清单）。
4. **核邮寄送达** → 须有邮寄凭证 + 签收记录，确认签收日期（实务送达审查表）。
5. **核公告送达** → 须在其他方式**无法送达时**方可公告；公告须载明完整内容；**公告满60日视为送达**（02_常见错误2.4；实务送达审查表"公告期限是否届满"）。
6. **核顺序一致** → 送达日期须与告知→决定→送达时序一致（调 [[程序违法排查]]）。
7. **超期办案预警** → 立案→决定应在法定期限内；听证/公告/监测/鉴定/送达时间**不计入**期限（评查细则决定书项"法定期限内作出"）；对比立案审批表与决定书/送达日期。
8. 命中问题 → 引用 [[评查细则全文2024指针]] / [[25项一票否决清单]] 第13/14项。

## 常见扣分点 / 易错点（Top 8）
- **🟠 送达回证缺失/无签收日期**：卷面基本要素扣2分（评查细则卷面要素送达回证项）。
- **🟠 留置送达无见证人/无录像**：送达瑕疵，高频扣分项（实务经验；高频扣分清单第二类第4项）。
- **🟠 公告送达未届满60日即视为送达**：法定条件不满足（02_常见错误2.4；实务送达审查表）。
- **🟠 公告送达在非其他方式不能送达时即适用**：顺序违法（02_常见错误2.4）。
- **🟡 邮寄送达无签收记录**：无法确认送达日期（实务送达审查表）。
- **🟡 直接送达非优先** / 代收人无资格：方式不合规。
- **🟡 当事人信息前后不一致影响送达**：文书瑕疵（高频扣分第三类第2项）。
- **🔴 超期作出决定/送达**（无扣除事由）：评查细则决定书项"法定期限内作出"；严重可构成重大程序问题（25项清单第14项"超期作出决定"）。
