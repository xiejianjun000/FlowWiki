## 何时读我（触发信号）
- 评查法律适用环节（合法性审查§4）
- 案卷结案日期在 2026-08-15 附近
- 处罚决定书引用法律名称/条文存疑
- 同一当事人有多份处罚决定，疑似重复罚款

## 目标拆解
法律适用 = 定性是否准、依据是否有效、引用是否到条款项。8-15 后引已废止10法 = 一票否决。
最终产出 = ⚖️ 法律适用核查结论（效力层 + 引用层 + 重复罚层 + 临界层）。

## 全流程步骤（逐步可跟）
1. **核临界日** → 先确认案卷**结案日期**（处罚决定书作出/送达日）：≥2026-08-15 → 强制适用法典（[[法典8-15施行倒计时看板]]第1242条）。
2. **核废止10法** → 8-15后结案案卷出现"环境保护法/环评法/海洋/大气/水/土壤/固废/噪声/放射性/清洁生产促进法"任一名 = 一票否决（评查细则§4重大质量问题第10项；25项清单第10项）。
3. **核引用层级** → 法条须引用到**条/款/项/目**具体层级，禁以"有关规定"笼统代替（评查细则§4第10项"未准确引用相应条款"；02_常见错误2.6(3)）。
4. **核定性相符** → 决定书中违法事实描述须与适用法条描述一致；已有暗管等证据不得降格认定为超标（评查细则§4第8项）。
5. **核兜底条款** → 无依据不得适用兜底/等外条款（评查细则§4第11项"无法定理由适用兜底条款"）。
6. **核从旧兼从轻** → 违法行为发生时旧法，处理时新法；新法处罚较轻或不认为违法的适用新法（法典§1057；评查细则§4第10项"违反从旧兼从轻"）。
7. **核一事不再罚** → 同一违法行为不得给予两次以上罚款（行政处罚法§29；评查细则§4第12项"加重/无理由减轻"关联）。
8. 命中问题 → 引用 [[评查细则全文2024指针]] / [[法典对照与废止10法]] 锁定。

## 常见扣分点 / 易错点（Top 8）
- **🔴 8-15后引已废止10法**：一票否决，处罚可被撤销（评查细则§4第10项；法典§1242）。
- **🔴 法条未引用到条款项**：仅写法律名称或"有关规定"（评查细则§4第10项；02_常见错误2.6(3)）。
- **🔴 定性错误/降格认定**：暗管偷排认定为超标排污（评查细则§4第8项）。
- **🔴 违反从旧兼从轻**：建设项目类未适用行为发生时法（评查细则§4第10项）。
- **🟠 无依据适用兜底/等外条款**：如查封扣押"其他造成污染"无法律支撑（评查细则§4第11项）。
- **🟠 一事二罚款**：同一行为重复罚款（行政处罚法§29）。
- **🟡 法律名称不规范**：用《反法》《水法》简称代替全称（02_常见错误2.6(2)）。
- **🟡 红头文件作处罚依据**：不得直接引用红头文件（02_常见错误2.6(1)效力层次）。
