1. **评查新案卷时**：先看07（评查细则全文），再对照21（处罚办法）、23（文书制作指南）

3. **法律适用审查**：2026年8月15日前用旧法合法，8月15日后引用已废止法律属重大问题（零分项）
4. **法典衔接**：重点看06（衔接指南）和17（法律责任编）
