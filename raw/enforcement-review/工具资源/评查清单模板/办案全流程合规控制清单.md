## 何时读我
- 办案人员自查：每个阶段是否合规、证据是否固化。
- 评查人员反查：案卷缺哪一步、对应哪个扣分项。
- 构建"执法履职免责清单"（SOUL 体系八·清单革命）：标准化防漏项。

> 本清单对应 SOUL v5.0 的"20章清单"思维，提炼为**10阶段打勾**。每阶段给出「必做动作 + 易漏项 + 评查关联」。引用 [[评查细则全文2024指针]] 具体项。

## 目标拆解
最终产出 = 一份可复制的✅打勾清单，漏项即预警，直接对接评查扣分项。

## 资料准备清单
- [ ] 本案卷宗（或实时办案记录）
- [ ] 评查细则2024（合法性/规范性项）
- [ ] 法典程序通则（§1052—1084）

## 全流程10阶段打勾清单

### 阶段1｜案前（线索/立案）
- [ ] 线索来源登记（信访/巡查/监测/移送）
- [ ] 管辖权核实（地域/级别/派出机构授权，法典§1062）
- [ ] 立案审批在时限内（发现线索→立案）
- ⚠️ 易漏：派出机构越权以自己名义立案

### 阶段2｜现场检查
- [ ] 不少于2名持证执法人员，笔录记证号
- [ ] 出示证件、表明身份
- [ ] 现场笔录记载异常信号（见 [[9种现场异常信号识别]]）
- ⚠️ 易漏：笔录未记证号且未附证复印件（合法性序号1隐患）

### 阶段3｜采样
- [ ] 一类污染物在车间排放口采样
- [ ] 采样记录/点位图/照片/视频齐备
- [ ] 保存运输条件合规、质控空白
- ⚠️ 易漏：采样过程无记录→证据不足（合法性序号3）

### 阶段4｜询问
- [ ] 开放式切入+细节钉死+闭环确认（见 [[企业博弈应对话术]]）
- [ ] 禁止诱导式发问，完整记录对方解释
- [ ] 关键陈述签字确认
- ⚠️ 易漏：只记有利陈述，笔录不完整

### 阶段5｜告知（处罚告知/听证告知）
- [ ] 五要素告知：事实/陈述申辩/听证/复议/诉讼
- [ ] 告知书送达回证完整
- ⚠️ 易漏：漏告听证权或复议诉讼期限（合法性序号5）

### 阶段6｜听证（应听必听）
- [ ] 符合听证条件(较大数额罚款等)已告知并组织
- [ ] 听证笔录入卷
- ⚠️ 易漏：应听未听 = 重大程序违法

### 阶段7｜法制审核
- [ ] 重大执法决定经法制审核
- [ ] 审核意见书面入卷
- ⚠️ 易漏：跳过审核或意见缺失

### 阶段8｜集体讨论
- [ ] 情节复杂/重大违法经集体讨论
- [ ] 讨论记录载明不同意见与结论
- ⚠️ 易漏：一人决定代替集体讨论

### 阶段9｜决定
- [ ] 案由写法典章节名（8-15后禁引废止法，见 [[法典对照与废止10法]]）
- [ ] 法律依据引具体条文号
- [ ] 裁量理由充分（见 [[湖南裁量基准指针]]）
- [ ] 救济途径与期限准确（复议60日/诉讼6个月）
- ⚠️ 易漏：决定书引用已废止法律 = 一票否决

### 阶段10｜送达
- [ ] 送达方式合法（直接/邮寄/公告）
- [ ] 送达回证签收日期/人明确
- [ ] 期限合规
- ⚠️ 易漏：送达回证瑕疵→程序违法隐患

## 常见漏项 Top 5
- ❌ 采样无记录（证据链第一环断）
- ❌ 告知漏听证权/期限
- ❌ 法制审核/集体讨论缺失
- ❌ 8-15后引废止法（红线）
- ❌ 裁量理由空白

## 输出成果模板
```
✅ 办案全流程合规自检
├── 阶段完成度：X/10
├── 已完成：[阶段列表]
├── 待办/风险：[漏项 + 对应评查扣分项/红线]
└── 履职免责结论：[清单完整=履职到位 / 缺项=补正建议]
```

【材料】<粘贴：当前阶段+已完成动作清单>
【输出】<✅勾稽表：已完成/待办/风险项 + 履职免责结论>
