# 生态环境监测条例施行要点

**施行时间：** 2026年1月1日
**立法意义：** 我国首部生态环境监测领域专门行政法规

---

## 📌 立法背景与意义

### 立法历程
- **2019年5月：** 生态环境监测司提出制订条例规划
- **2025年5月：** 列入国务院年度立法计划
- **2025年10月17日：** 国务院第70次常务会议审议通过
- **2026年1月1日：** 正式施行

### 立法意义
1. **填补立法空白：** 生态环境监测领域首部专门行政法规
2. **解决核心问题：** 监测碎片化、数据质量难管控、主体责任不清晰
3. **筑牢数据生命线：** 为生态文明建设提供数据支撑
4. **强化社会监管：** 加强对社会监测机构的监管，规范监测市场

## 🔴 核心要点解读

### 一、监测格局构建

#### 政府主导、部门协同、企业履责、社会参与

1. **政府主导：** 
   - 统一规划监测网络
   - 组织开展质量监测、监督监测
   - 建立风险监测预警体系

2. **部门协同：** 
   - 生态环境主管部门牵头
   - 其他部门配合

3. **企业履责：** 
   - 排污单位开展自行监测
   - 安装自动监测设备
   - 保存原始监测记录

4. **社会参与：** 
   - 技术服务机构提供监测服务
   - 公众参与监督

### 二、自行监测要求（重点）

#### 排污单位义务
1. **制定监测方案：** 按照生态环境监测有关规范和标准制定
2. **安装视频监控：** 主要监测点位安装、使用视频监控设备
3. **使用合规设备：** 使用符合国家标准和规范的监测设备
4. **建立质量管理制度：** 建立健全监测数据质量管理制度
5. **保存原始记录：** 保存原始监测记录
6. **禁止弄虚作假：** 不得以任何方式对监测数据弄虚作假

#### 法律后果
- 未按规定开展自行监测 → 行政处罚
- 监测数据弄虚作假 → 从重处罚，可能移送刑事

### 三、技术服务机构监管（重点）

#### 备案制度
- 向生态环境主管部门备案
- 具备相应的设施设备、技术能力、技术人员和管理能力

#### 行为规范
1. **独立、客观、公正：** 独立开展监测服务，不受干扰
2. **利益冲突回避：** 不得同时接受可能存在利益冲突的委托
3. **保存相关材料：** 按规定保存监测数据、记录等相关材料
4. **全过程可追溯：** 保证业务活动全过程可追溯

#### 法律责任（第43条）
技术服务机构开展监测服务不遵守生态环境监测规范或者标准，导致监测数据失真的：
- 没收违法所得
- 罚款
- 情节严重的，停业整顿

### 四、监测数据法律地位

#### 作为执法依据
- 环境保护税征收
- 生态环境统计
- 行政执法
- 行政处罚

#### 证据效力
- 监测数据具有法律效力
- 可作为行政处罚、刑事案件的证据
- 需符合法定程序和技术规范

### 五、信用评价制度

#### 分级分类监管
- 建立生态环境监测信用评价制度
- 对技术服务机构实行分级分类监管
- 信用评价结果向社会公开

#### 行业自律
- 加强行业自律管理
- 制定行业规范
- 开展行业培训


## 🟠 对执法工作的影响

### 一、执法依据更充分

#### 法律依据
- **《生态环境监测条例》第43条：** 技术服务机构违法
- **《生态环境监测条例》第45条：** 弄虚作假
- 配合《大气污染防治法》《水污染防治法》等适用

### 二、证据要求更严格

#### 证据合规性要求
1. **人员要求：** 执证执法人员在场
2. **样品处理：** 样品必须密封
3. **分析方法：** 必须符合技术规范
4. **过程记录：** 详细记录采样过程（拍照、录像）

#### 证据链完整
- 现场笔录 + 监测报告 + 原始记录 + 视频监控
- 形成完整证据链

### 三、处罚力度更大

#### 行政处罚
- 没收违法所得
- 罚款（最高可达数十万元）
- 停业整顿

#### 刑事移送
- 情节严重的，涉嫌提供虚假证明文件罪
- 《刑法》第229条

### 四、监管范围更广

#### 从排污单位扩展到技术服务机构
- 排污单位自行监测
- 技术服务机构监测服务
- 自动监测设备运维单位


## 📋 案卷评查新增要点

### 一、自行监测案卷

#### 审查内容
1. **监测方案制定：** 是否按照规范制定监测方案
2. **视频监控安装：** 主要监测点位是否安装视频监控
3. **设备合规性：** 监测设备是否符合国家标准
4. **质量管理制度：** 是否建立监测数据质量管理制度
5. **原始记录保存：** 是否保存原始监测记录
6. **数据真实性：** 监测数据是否弄虚作假

### 二、技术服务机构案卷

#### 审查内容
1. **备案情况：** 是否向生态环境主管部门备案
2. **技术能力：** 是否具备相应设施设备、技术能力、技术人员
3. **利益冲突：** 是否同时接受可能存在利益冲突的委托
4. **材料保存：** 是否按规定保存监测数据、记录
5. **规范遵守：** 是否遵守生态环境监测规范和标准
6. **数据失真：** 监测数据是否失真

### 三、采样程序案卷

#### 审查内容
1. **执法人员在场：** 执证执法人员是否在场
2. **样品密封：** 样品是否密封
3. **分析方法合规：** 分析方法是否符合技术规范
4. **过程记录完整：** 采样过程是否有详细记录（拍照、录像）


## ⚠️ 风险提示

### 高败诉风险点

1. **采样程序不规范：** 
   - 执法人员未在场 → 程序违法
   - 样品未密封 → 证据无效
   - 分析方法不符规范 → 监测数据无效

2. **证据链不完整：**
   - 仅凭监测报告 → 证据不足
   - 未调取原始记录 → 数据真实性存疑
   - 未保存视频监控 → 无法证明采样过程

3. **法律适用错误：**
   - 未引用《生态环境监测条例》
   - 应移送刑事未移送

### 应对措施

1. **规范采样程序：** 
   - 执法人员在场
   - 样品密封
   - 详细记录

2. **完整证据链：**
   - 现场笔录
   - 监测报告
   - 原始记录
   - 视频监控

3. **准确适用法律：**
   - 行政处罚 + 刑事移送
   - 区分违法情节
