# 实务经验 - 2026年6月28日

## 一、案卷评查组织模式创新

### 1. 贺州市"显微式解剖"评查模式

**评查组织**：
- **模式**："县区交叉互评、市局法制骨干复核"
- **评查范围**：2026年第一季度已制发行政处罚决定书的15个案卷
- **覆盖领域**：大气污染、水污染、固体废物管理等多个生态环境重点监管领域

**评查方法**：
- 对照《生态环境行政执法案卷评查细则》
- 紧扣合法性、规范性两大核心评查标准
- 从7个方面逐卷逐页"过筛子"：
  1. 执法主体
  2. 事实证据
  3. 法律适用
  4. 程序合法
  5. 自由裁量
  6. 文书制作
  7. 案卷归档

**整改机制**：
- 第一时间梳理汇总问题
- 深入剖析问题根源
- 结合执法工作实际制定针对性整改措施
- 推动各类问题整改落地落细、闭环清零
- **成效**：所有问题已全部整改到位，新增案件规范化水平显著提升，同类问题发生率大幅下降

**评查启示**：
1. **交叉评查优势**：避免"自己查自己"的盲区，提高评查客观性
2. **显微式解剖**：不放过任何细节，逐页逐行检查
3. **闭环整改**：不只评查，更重视整改落实和效果跟踪
4. **量化成效**：用"同类问题发生率"等指标检验评查效果

### 2. 张掖高台分局案卷自查评查经验

**评查原则**：
- 问题导向
- 逐项核查
- 从严把关
- 精益求精

**重点查摆问题**：
1. 文书填写表述不严谨
2. 案卷附件归档不细致
3. 签章流程排序不规范

**工作机制**：
- 建立问题台账清单
- 实行逐项认领
- 分类汇总
- 溯源分析
- 深入剖析问题产生根源
- 精准明确整改重点和优化方向

**评查启示**：
1. **问题分类**：将问题分为"文书表述""归档规范""签章流程"等类别，便于针对性整改
2. **溯源分析**：不只看表面问题，更要分析问题的根源
3. **台账管理**：建立问题台账，实行清单化管理

## 二、案卷评查常见问题类型（2026年最新）

根据多地评查情况，整理常见问题的新表现：

### 1. 文书制作类问题

- **表述不严谨**：文书中事实描述、法律依据引用表述不够准确
- **填写不规范**：文书必填项漏填、填写不完整
- **逻辑不严密**：文书之间时间逻辑、事实逻辑不一致

### 2. 程序规范类问题

- **签章流程排序不规范**：文书签章、审批流程顺序不符合要求
- **时间节点不合规**：调查取证、审理审核、决定送达等时间节点不符合法定程序
- **权利告知不完整**：未充分保障当事人陈述申辩、听证等权利

### 3. 案卷归档类问题

- **附件归档不细致**：案卷附件材料不完整、排序混乱
- **材料缺失**：现场检查笔录、询问笔录、监测报告等关键证据缺失
- **归档不及时**：案件办结后未及时整理归档

## 三、评查技巧提升

### 1. "显微式"评查法

**操作步骤**：
1. **全面浏览**：先快速浏览整套案卷，了解案件基本情况
2. **重点突破**：聚焦执法程序、事实认定、法律适用等关键环节
3. **逐页核查**：逐卷逐页"过筛子"，不放过任何细节
4. **对标对表**：严格对照评查细则，逐项检查打分

### 2. "七维度"评查框架

从7个维度系统评查案卷质量：
1. **执法主体**：执法主体资格、执法人员资格
2. **事实证据**：违法事实认定、证据收集固定
3. **法律适用**：法律条款引用、裁量基准适用
4. **程序合法**：执法程序、审批流程
5. **自由裁量**：裁量基准、处罚幅度
6. **文书制作**：文书格式、表述规范
7. **案卷归档**：归档完整、排序规范

### 3. "闭环式"整改跟踪

1. **问题发现**：评查时详细记录问题
2. **问题反馈**：及时向办案机构反馈
3. **整改落实**：跟踪整改进度
4. **效果检验**：复查整改效果
5. **能力提升**：总结经验，提升办案水平

## 四、评查工作建议

1. **建立评查人才库**：选拔执法和案件审查骨干，建立专业化评查队伍
2. **定期开展交叉评查**：避免"自己查自己"，提高评查客观性
3. **强化评查结果运用**：将评查结果与执法质量考核挂钩
4. **注重评查经验总结**：定期汇编评查典型案例，供学习借鉴
5. **加强评查业务培训**：提升评查人员专业能力和水平
