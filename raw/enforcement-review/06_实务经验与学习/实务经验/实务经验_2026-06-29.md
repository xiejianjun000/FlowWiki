# 实务经验 2026-06-29

## 一、证据规则实务要点

### （一）行政处罚证据种类
根据《行政处罚法》第三十三条，证据包括：
1. 书证
2. 物证
3. 视听资料
4. 电子数据
5. 证人证言
6. 当事人的陈述
7. 鉴定意见
8. 勘验笔录、现场笔录

### （二）取证时限要求
- 取证时限：作出行政行为之前
- 行政行为作出后自行收集的证据，不能作为行政行为合法的依据
- 行政相对人和利害关系人提供证据不在此限

### （三）证据形式要件

#### 书证
1. 应提取原件并由原件持有人签章
2. 提取复制件、影印件、抄录件的，应注明出处，经核对无异后加盖印章
3. 报表、图纸、会计账册、专业技术资料等，应附说明材料
4. 询问笔录、陈述申辩笔录、听证笔录必须经笔录当事人签章
5. 当事人拒绝签章的，由笔录人在笔录上注明原因

#### 物证
1. 应提取原物
2. 确有困难的，可提取与原物核对无误的复制件或照片、录像

#### 现场笔录
- **证据采信标准**：
  - 符合制作规范、内容真实、程序合法的现场笔录，直接作为定案依据，证明力高于其他书证
  - 无执法人员签名、无见证人见证、当事人拒绝签名且未注明原因、事后补记的，不予采信
  - 现场笔录与其他证据矛盾的，结合全案证据综合认定
  - 当事人能提供充分反证证明内容虚假的，不予采信

### （四）非法证据排除
以下证据不能作为定案依据：
1. 严重违反法定程序收集的证据材料
2. 以偷拍、偷录、窃听等手段获取侵害他人合法权益的证据材料
3. 以利诱、欺诈、胁迫、暴力等不正当手段获取的证据材料
4. 不能正确表达意志的证人提供的证言
5. 不具备合法性和真实性的其他证据材料

### （五）证据先行登记保存注意事项
1. 应事先获得行政负责人批准，情况紧急的应在24小时内补办手续
2. 必须在7日内予以处理
3. 严格适用条件：证据可能灭失或以后难以取得
4. 不能变相以证据登记保存替代查封、扣押

### （六）案卷评查证据审查要点
1. **证据完整性**：证据链是否完整，能否形成闭环
2. **证据合法性**：取证程序是否合法，是否存在非法证据
3. **证据关联性**：证据与违法事实是否关联
4. **证据真实性**：证据是否真实可靠
5. **证据充分性**：证据是否足以认定违法事实

## 二、自由裁量权适用实务

### （一）裁量基准计算公式（山东省示例）

**公式**：X = M × A × (1 + ΣB_i / n)

其中：
- X：处罚金额
- M：法定处罚金额上限
- A：首要裁量因素裁量等级
- B：其余裁量因素裁量等级
- n：其余裁量因素个数

### （二）裁量因素审查要点
1. **违法情节**：持续时间、影响范围、危害后果等
2. **当事人情况**：是否初犯、主观过错程度、配合调查情况
3. **整改情况**：是否及时改正、整改效果
4. **法定情节**：是否有从轻、减轻、不予处罚情节

### （三）从轻、减轻处罚适用条件
根据《行政处罚法》：
1. 主动消除或者减轻违法行为危害后果的
2. 受他人胁迫或者诱骗实施违法行为的
3. 主动供述行政机关尚未掌握的违法行为的
4. 配合行政机关查处违法行为有立功表现的
5. 法律、法规、规章规定其他应当从轻或者减轻行政处罚的

### （四）不予处罚适用条件
1. 违法行为轻微并及时改正，没有造成危害后果的
2. 当事人有证据足以证明没有主观过错的
3. 法律、法规、规章规定其他不予行政处罚情形

### （五）案卷评查裁量审查要点
1. **裁量依据**：是否适用正确的裁量基准
2. **裁量理由**：是否说明裁量理由
3. **裁量幅度**：处罚幅度是否在裁量基准范围内
4. **特殊情况**：从轻、减轻、不予处罚是否符合法定条件

## 三、现场笔录制作规范

### （一）制作要求
1. 由行政执法人员制作
2. 如实记录现场情况
3. 由执法人员、当事人签名或盖章
4. 当事人拒绝签名的，应注明原因
5. 有见证人的，见证人应签名或盖章

### （二）常见问题
1. 无执法人员签名
2. 当事人拒绝签名未注明原因
3. 事后补记
4. 内容与其他证据矛盾
5. 无见证人见证

### （三）案卷评查审查要点
1. 是否当场制作
2. 是否由执法人员签名
3. 当事人拒绝签名的，是否注明原因
4. 是否有见证人见证（必要时）
5. 内容是否真实、完整

## 五、案卷评查常见问题清单

### （一）程序类问题
1. 立案审批不及时
2. 调查取证程序不规范
3. 告知程序缺失或瑕疵
4. 听证程序不规范
5. 审批程序不完整
6. 送达程序不规范
7. 执行程序不规范

### （二）证据类问题
1. 证据收集不全面
2. 证据形式不规范
3. 证据链不完整
4. 非法证据未排除
5. 证据采信依据不足

### （三）法律适用类问题
1. 法律依据援引错误
2. 法律条款适用错误
3. 法律更新未及时适用
4. 法律竞合处理不当

### （四）裁量类问题
1. 裁量依据缺失
2. 裁量理由不充分
3. 裁量幅度不当
4. 从轻、减轻处罚理由不足

### （五）文书类问题
1. 文书格式不规范
2. 文书内容不完整
3. 文书逻辑不清晰
4. 文书签名盖章不规范
5. 文书日期矛盾


## 学习总结

### 证据规则核心要点
1. 取证必须在行政行为作出前完成
2. 证据必须符合形式要件要求
3. 非法证据必须排除
4. 现场笔录证明力高于其他书证

### 裁量权适用核心要点
1. 严格适用裁量基准
2. 说明裁量理由
3. 从轻、减轻、不予处罚需符合法定条件

### 法典施行准备工作
1. 更新法律依据
2. 调整评查标准
3. 完善文书模板
4. 开展培训学习
