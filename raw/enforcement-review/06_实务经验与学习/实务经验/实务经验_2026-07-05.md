# 实务经验 - 2026年7月5日

## 一、案卷评查实务技巧更新

### （一）评查新模式学习

#### 1. "显微式解剖"评查法

**来源**: 贺州市生态环境局2026年第一季度案卷交叉评查

**核心方法**
- 对案件开展"显微式解剖"
- 全面排查梳理执法工作短板与薄弱环节
- 聚焦执法程序、事实认定、法律适用等关键环节

**实操步骤**
1. **程序审查**: 立案、调查、告知、决定、送达等各个环节
2. **事实审查**: 违法事实是否清楚、证据是否充分
3. **法律审查**: 法律适用是否正确、处罚幅度是否合理
4. **文书审查**: 文书制作是否规范、表述是否准确

**评查要点**
- 不遗漏任何一个细节
- 不放过任何一个疑点
- 深入剖析、细致审查

**适用案件**
- 复杂案件
- 争议案件
- 新型案件

---

#### 2. "一卷一评"模式

**来源**: 济南市2026年第一次案卷交流互评活动

**核心方法**
- 逐卷填写评审意见表
- 对典型问题和常见错误集中剖析交流
- 达到纠偏补弱、规范引领的效果

**实操步骤**
1. **分组评查**: 评查人员分成若干小组
2. **逐卷评查**: 对每一份案卷进行细致审核
3. **逐项打分**: 对照评查细则逐项打分
4. **记录问题**: 对存在的问题详细记录
5. **集中研讨**: 对疑点难点问题集中剖析交流

**评查要点**
- 对照评查细则
- 重点围绕六大核心要素：
  1. 执法主体是否适格
  2. 违法事实是否清楚
  3. 证据是否充分
  4. 法律适用是否正确
  5. 执法程序是否合法
  6. 文书制作是否规范

**适用案件**
- 所有类型案件
- 特别适合评查人员培训

#### 3. "县区交叉互评+市局法制骨干复核"模式

**来源**: 贺州市生态环境局

**核心方法**
- 县区交叉互评：避免本地保护主义
- 市局法制骨干复核：确保评查质量

**实操步骤**
1. **交叉分配**: 甲县评乙县、乙县评丙县、丙县评甲县
2. **独立评查**: 交叉评查人员独立评查
3. **初步打分**: 交叉评查人员给出初步评分
4. **法制复核**: 市局法制骨干复核评分
5. **最终定级**: 综合交叉评查和法制复核确定最终等级

**评查要点**
- 交叉评查避免人情干扰
- 法制复核确保专业质量
- 双重把关提高评查公信力

**适用案件**
- 所有类型案件
- 特别适合大范围评查活动

**来源**: 法意科技行政执法案卷评查系统

**核心方法**
- 人工智能自动校验
- 人工复核把关

**AI自动评查功能**

   - 编码规范性

   - 审批手续完整性

3. **实体方面评查**
   - 自由裁量合法性
   - 法律依据引用准确性
   - 处罚幅度合理性

4. **文书质量评查**
   - 文书内容完整性
   - 文书制作规范性
   - 文书逻辑性
   - 文字错误检查

**适用场景**

### （三）案卷审阅技巧

#### 1. 从内部案卷找漏洞翻盘

**来源**: 《第79篇行政处罚案卷审阅技巧》

**核心方法**
- 企业申请阅卷的法定权利
- 从内部案卷找漏洞
- 找到漏洞即可翻盘

**企业阅卷权利**
- 案件进入申辩、听证、复议、诉讼阶段
- 企业有权依法申请查阅、复制全部执法案卷材料
- 执法机关不得拒绝、隐瞒案卷内容
- 完整阅卷是企业维权的基础

**案卷重点核查四大板块**

**一、程序文书**
- 立案文书是否齐全
- 审批文书是否齐全
- 送达文书是否齐全
- 告知文书是否齐全
- 文书是否合法

**审查要点**
1. 文书是否缺失
2. 文书是否合法
3. 文书是否规范
4. 文书是否及时

**常见问题**
- 立案审批表缺失
- 送达回证缺失
- 告知文书缺失
- 审批手续不全


**二、证据材料**
- 证据是否真实
- 证据是否完整
- 证据是否有效
- 证据是否相互印证

**审查要点**
1. 证据三性：真实性、合法性、关联性
2. 证据链闭环
3. 证据相互印证
4. 证据充分性

**常见问题**
- 证据造假
- 证据不足
- 证据矛盾
- 证据链断裂


**三、笔录内容**
- 笔录是否篡改
- 笔录是否遗漏
- 笔录是否与事实相符

**审查要点**
1. 笔录真实性
2. 笔录完整性
3. 笔录一致性
4. 笔录与其他证据印证

**常见问题**
- 笔录篡改
- 笔录遗漏关键信息
- 多份笔录矛盾
- 笔录与事实不符


**四、适用法条**
- 法条适用是否准确
-. 处罚幅度是否合规

**审查要点**
1. 法律条文是否正确
2. 法律条款是否对应
3. 处罚幅度是否合理
4. 裁量理由是否充分

**常见问题**
- 法条套用错误
- 条款引用错误
- 处罚幅度不当
- 裁量理由不足


**案卷漏洞的核心翻盘思路**

**一票否决类漏洞**
1. 案卷文书缺失（程序违法）
2. 前后矛盾（事实不清）
3. 程序倒置（程序违法）
4. 证据造假（证据无效）
5. 法条套用错误（适用法律错误）
6. 审批瑕疵（程序违法）

**翻盘策略**
- 只要发现任意一项实质性漏洞
- 即可主张处罚违法、证据无效
- 申请撤销处罚、改判结果


#### 2. 阅卷审查的基本方法

**来源**: 百度文库《阅卷审查方法》

**适用范围**
- 适用于犯罪次数较多的案件
- 适用于多人犯罪比较复杂的案件

**方法一: 先证后供法**

**适用案件**
- 犯罪次数较多
- 延续时间较长
- 不同性质犯罪交叉
- 犯罪嫌疑人记不清每次作案详细情况

**操作步骤**
1. 首先阅读被害人陈述、证人证言、鉴定结论等证据材料
2. 将同一起案件的证据材料按照与案件事实之间关联性大小顺序排列在一起
3. 分析能否形成证据体系
4. 再按照作案时间顺序将不同案件的证据材料进行排列
5. 制作取证情况一览表
6. 对照证据材料在口供材料中进行查找
7. 分析犯罪嫌疑人对此有无供述
8. 供述与其它证据之间是否一致、能否相互印证

**注意要点**
- 不能从所有细节上强求供证一致
- 只要有几个关键点能相互印证即可认定案件事实


**方法二: 共同犯罪案件阅卷方法**

**适用案件**
- 多人犯罪
- 共同犯罪
- 主犯从犯需要区分

**操作步骤**
1. 分别审查各犯罪嫌疑人的供述
2. 审查犯罪嫌疑人之间相互推诿责任的情况
3. 分析各犯罪嫌疑人在犯罪中的地位、作用
4. 对照其他证据认定主犯、从犯

**注意要点**
- 共同犯罪人之间往往避重就轻
- 相互推诿责任
- 需要细致审查、综合判断

## 二、程序法实务解读

### （一）行政复议法实施条例（2026年7月1日起施行）

#### 1. 全面审查要求

**法律要求**
- 行政复议机关应当对行政行为的**合法性、适当性进行全面审查**
- 不仅审查合法性，还审查适当性

**对案卷评查的影响**
1. **合法性审查**: 案卷中已有体现
2. **适当性审查**: 案卷中需要补充体现

**适当性审查内容**
- 裁量理由是否充分
- 过罚相当原则是否落实
-. 主观过错是否考虑
- 危害后果是否考虑
- 整改情况是否考虑

**案卷补充要求**
1. 裁量理由说明书
2. 说理式执法文书
3. 集体讨论记录（重大复杂案件）
4. 听取当事人陈述申辩记录


#### 2. 行政复议范围细化

**新增可复议事项**
- 对列入**严重失信主体名单决定**不服
- 对**失信惩戒措施**不服

**评查要点**
1. 严重失信主体名单决定是否合法
2. 失信惩戒措施是否适当
3. 列入严重失信主体名单的程序是否合法


#### 3. 行政复议前置情形

**适用行政复议前置的情形**
1. 对当场作出的行政处罚决定不服
2. 未履行法定职责
3. 不予公开政府信息
4. 其他法律规定复议前置的情形

**评查要点**
1. 处罚决定书是否告知复议前置
2. 当事人是否先申请复议
3. 复议机关是否依法受理


#### 4. 提级审理和合并审理

**提级审理**
- 明确提级审理的具体情形和程序
- 重大复杂案件提级审理

**合并审理**
- 增加合并审理制度
- 同类案件合并审理

**评查要点**
1. 重大复杂案件是否提级审理
2. 同类案件是否合并审理
3. 提级审理程序是否合法
4. 合并审理程序是否合法


### （二）行政诉讼法实务

#### 1. 起诉期限体系

**一般起诉期限**
- 自知到具体行政行为之日起**6个月内**

**最长起诉期限**
- **5年**: 涉及不动产的案件
- **20年**: 因不动产提起诉讼的案件

**评查要点**
1. 处罚决定书起诉期限告知是否准确
2. 是否告知6个月起诉期限
3. 是否告知5年/20年最长起诉期限


#### 2. 复议与诉讼衔接

**复议后起诉期限**
- 不服复议决定: 收到复议决定书之日起**15日内**
- 复议机关逾期不作出复议决定: 复议期满之日起**15日内**

**评查要点**
1. 处罚决定书是否告知复议后起诉期限
2. 是否告知15日起诉期限
3. 是否告知复议机关逾期不决定的起诉期限
