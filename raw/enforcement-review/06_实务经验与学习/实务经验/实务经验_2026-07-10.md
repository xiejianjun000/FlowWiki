# 实务经验与技术要点（2026-07-10更新）

> 来源：济南市、定州市、黑龙江省等案卷评查实践经验
> 整理定位：案卷评查操作规范 + 证据链构建 + 评查流程优化

---

## 一、案卷评查流程规范

### 1.1 评查准备阶段

| 步骤 | 内容 | 要点 |
|------|------|------|
| 组建评查组 | 业务骨干 + 法律顾问 + 专家 | 引入外部专业视角 |
| 培训先行 | 学习评查细则 + 案例剖析 | 统一评分标准 |
| 工具准备 | 评审意见表 + 评分细则 | 标准化操作 |
| 案卷选取 | 随机抽取 + 选送相结合 | 保证覆盖面 |

### 1.2 评查实施阶段

**核心六要素审查：**
1. **执法主体是否适格** → 核查执法证、授权文件
2. **违法事实是否清楚** → 事实描述是否完整、明确
3. **证据是否充分** → 证据链是否闭环
4. **法律适用是否正确** → 法条引用准确性（含法典衔接）
5. **执法程序是否合法** → 时限、告知、送达全流程
6. **文书制作是否规范** → 格式、内容、签章

### 1.3 评查反馈阶段

| 动作 | 目的 |
|------|------|
| 逐卷填写评审意见表 | 问题可追溯 |
| 集中剖析典型问题 | 经验共享 |
| 现场通报存在问题 | 及时整改 |
| 落实整改责任 | 闭环管理 |

## 二、证据链构建要点

### 2.1 监测数据类证据审查

**🔴 重点核查清单：**

| 核查项 | 异常信号 | 评查动作 |
|--------|----------|----------|
| 原始记录 vs 报告 | 数据不一致 | 追溯原始记录 |
| 采样时长 | 报告显示时长与实际不符 | 核查设备日志 |
| 采样小票文件号 | 不同文件号数据雷同 | 核对编号规则 |
| 工况/标况体积 | 异常一致或异常波动 | 技术分析 |
| 可编辑文档 | 发现篡改软件/工具 | 扣押取证 |

### 2.2 在线监测数据异常识别

**典型异常模式：**

| 异常类型 | 特征表现 | 证据类型 |
|----------|----------|----------|
| 流量与浓度不匹配 | 白天流量为0但浓度有波动 | 平台数据 |
| 数据骤降 | 超标后骤降至接近零 | 历史数据曲线 |
| 停运期间有数据 | 设备离线但平台有数据 | 平台日志 |
| 维护状态异常 | 标记维护但工况正常 | 运维记录 |

### 2.3 第三方监测报告审查

**审查要点：**
1. 资质认定证书有效性
2. 人员持证情况
3. 设备检定/校准证书
4. 原始记录完整性
5. 报告与记录一致性
6. 数据计算准确性


## 三、法律适用准确性审查

### 3.1 法典衔接审查（8月15日后重点）

| 审查阶段 | 内容 |
|----------|------|
| **现状** | 核查是否仍引用已废止法律（旧环境保护法、水/大气/土壤/固废/噪声污染防治法等10部） |
| **过渡期** | 8月15日前完成的案件，确保法律适用正确 |
| **倒计时** | 8月15日后结案案件，必须引用生态环境法典 |

### 3.2 裁量说理充分性审查

**评查标准：**

| 必要要素 | 审查内容 |
|----------|----------|
| 事实认定 | 违法事实描述是否完整 |
| 性质判断 | 违法行为的法律性质 |
| 情节考量 | 从重/从轻/减轻情节是否考虑 |
| 社会危害 | 危害后果是否评估 |
| 量罚梯度 | 是否对应裁量基准的具体档次 |
| 说理逻辑 | 事实→性质→情节→危害→量罚是否贯通 |

**⚠️ 评查警示**：仅写"依据裁量基准第X条罚款Y元"属**说理不充分**，应按评查细则对应条款扣分。


## 四、程序合法性审查

### 4.1 告知权利完整性审查

| 告知项 | 法定要求 | 评查要点 |
|--------|----------|----------|
| 陈述申辩权 | 处罚告知书必须告知 | 是否写明期限（5日/7日） |
| 听证权 | 符合听证条件的必须告知 | 重大处罚须告知听证权 |
| 复议权 | 必须告知 | 期限60日（一般）/15日（复议前置） |
| 诉权 | 必须告知 | 期限**六个月**（新法要求） |

### 4.2 送达程序审查

| 送达方式 | 证据要求 | 评查要点 |
|----------|----------|----------|
| 直接送达 | 送达回证（签收日期） | 核对日期一致性 |
| 留置送达 | 见证人/拍照录像 | 是否符合法定条件 |
| 邮寄送达 | 邮寄凭证+签收记录 | 签收日期确认 |
| 公告送达 | 公告材料+期满证明 | 公告期限是否届满 |


## 五、评查质量提升策略

### 5.1 评查模式创新

| 模式 | 做法 | 价值 |
|------|------|------|
| 交叉互评 | 不同地区/部门互查 | 引入外部视角，发现内部盲区 |
| 法律顾问参与 | 律师/专家加入评查组 | 提升法律专业性 |
| 初评+复审 | 骨干初评+专家复审 | 保证准确性 |
| 以评带训 | 边评查边培训 | 能力提升同步 |

### 5.2 评查结果运用

1. **问题整改**：逐项记录→归类梳理→落实责任→限期整改
2. **经验推广**：创新做法→典型示范→复制推广
3. **能力建设**：共性问题→培训需求→课程开发
4. **机制完善**：系统问题→制度改进→流程优化


## 六、常见错误与改进建议

### 6.1 高频扣分项

| 扣分类型 | 典型表现 | 改进建议 |
|----------|----------|----------|
| 法律适用错误 | 引用已废止法律 | 立即更新法规库 |
| 裁量说理不足 | 仅写条款不说明 | 强化说理逻辑 |
| 送达不规范 | 送达日期不一致 | 规范送达流程 |
| 告知遗漏 | 未告知诉权/复议权 | 完善告知模板 |
| 证据链断裂 | 单一证据定案 | 构建完整证据链 |

### 6.2 评查效率提升

| 技巧 | 说明 |
|------|------|
| 清单式评查 | 逐项勾选，提高效率 |
| 批量比对 | 利用技术手段比对数据一致性 |
| 模板化反馈 | 标准化评审意见表 |
| 案例库积累 | 建立典型问题案例库 |
