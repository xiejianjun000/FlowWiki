# 案卷评查实务经验（2026-06-27）

## 一、各地案卷评查创新实践

### （一）贺州市：显微式解剖评查模式

**评查时间**：2026年第一季度

**评查模式**：
- 采取"县区交叉互评、市局法制骨干复核"的模式
- 对15个已制发行政处罚决定书的案卷进行"全面体检"
- 覆盖大气污染、水污染、固体废物管理等多个生态环境重点监管领域

**评查方法**：
- 严格对照《生态环境行政执法案卷评查细则》
- 紧扣合法性、规范性两大核心评查标准
- 从执法主体、事实证据、法律适用、程序合法、自由裁量、文书制作、案卷归档等**7个方面**逐卷逐页"过筛子"
- 对案件开展"显微式解剖"，全面排查梳理生态环境执法工作短板与薄弱环节

**整改机制**：
- 针对评查发现的问题，第一时间梳理汇总、深入剖析问题根源
- 结合执法工作实际制定针对性整改措施
- 推动各类问题整改落地落细、闭环清零
- 目前所有问题已全部整改到位，新增案件规范化水平显著提升，同类问题发生率大幅下降

**评查启示**：
1. "显微式解剖"强调细致入微的评查态度
2. 7个方面全覆盖评查确保无遗漏
3. 闭环整改机制确保问题真正解决
4. 整改效果跟踪验证（同类问题发生率下降）

### （二）赤峰市：两轮交叉+集中现场评查模式

**评查时间**：2026年度第一次案卷评查

**评查范围**：
- 覆盖全市各旗县区生态环境执法机构
- 共调取2025年11月份以来共60余卷案卷
- 涉及大气、水、固废等多个重点执法领域

**评查模式创新**：
- **创新采用"两轮旗县区交叉互评、一轮集中现场评查"相结合的模式**
- 首轮交叉互评由各旗县区执法骨干组成2个互评小组
- 按照《生态环境行政执法案卷评查细则》，从执法主体、事实证据、法律适用、程序规则、文书制作等方面逐卷进行**两轮次的交叉评查**
- 确保评查尺度统一、标准严格
- 随后组织全体参评人员开展**集中现场评查**，现场汇总两轮交叉互评意见
- 对疑难争议问题进行**集体研讨和定性**

**专家点评机制**：
- 特别邀请了**自治区生态环境综合行政执法总队案卷评查专家**对评查结果进行点评
- 专家结合典型案例，深入剖析了案卷中易发、多发的程序瑕疵、文书表述不规范等问题
- 提出了针对性改进建议

**培训效果**：
- 共有40名一线执法及法制审核人员全程参与
- 通过**评训结合**的方式，既发现了共性问题，也推广了优秀案卷的示范经验

**结果运用**：
- 根据评查通报，督促各旗县区限期整改
- 将评查结果**纳入年度执法考核**
- 持续提升生态环境执法规范化、精细化水平

**评查启示**：
1. "两轮交叉+集中现场"模式提高评查准确性
2. 专家点评提升评查专业性
3. 评训结合实现"以评代训"
4. 结果纳入考核强化约束力

### （三）新余市："一案双评、交叉互评"模式

**评查时间**：2026年首次案卷评查（1-2月案卷）

**评查组织**：
- 采取"一案双评、交叉互评"的方式
- 对全市生态环境执法办案情况进行全面"把脉问诊"
- 由各县(区)大队及直属大队共同参与
- 共组建5个评查组，抽调18名执法业务骨干

**评查标准**：
- 严格依据《生态环境行政执法案卷评查细则(2024年版)》
- 围绕执法的合法性与规范性
- 重点从事实认定、证据链条、法律适用、程序合规、自由裁量、文书制作等核心要素入手
- 逐项核查、量化打分
- 确保标准统一、尺度一致、结果客观

**复核机制**：
- 新余市生态环境保护综合执法支队对案卷评查结果进行了**统一复核**
- 总体来看，全市生态环境执法案卷质量良好，展现了办案人员严谨规范、精准敏锐的专业素养

**持续改进**：
- 将以此次交叉评查为契机
- 持续开展案卷评查、业务培训和岗位练兵活动
- 深入推进严格规范公正文明执法
- 不断提升执法队伍的专业化、规范化、精细化水平

**评查启示**：
1. "一案双评"确保评查质量
2. 统一复核机制避免尺度不一
3. 持续评查+培训+练兵组合推进


### （四）济南市：四轮次全覆盖交流互评模式

**评查时间**：2026年分四轮次推进

**评查组织**：
- 市生态环境保护综合行政执法支队组织开展2026年案卷交流互评活动
- **分四轮次推进**，每轮组织2至4个区县分局的案卷进行集中互评
- 根据各区县分局案卷数量合理调配，**实现全市全覆盖**

**评查参与**：
- 2026年4月组织开展了全市第一次行政执法案卷交流互评活动
- 全市12个区县分局共派出38名骨干人员参加
- 市生态环境保护综合行政执法支队派出6名骨干全程参与
- 每轮邀请其他两个区县分局各1至2名业务骨干、有关法律顾问参与交叉互评

**评查范围**：
- 共互评行政执法案卷100余份
- 案件类型涵盖**行政处罚、不予行政处罚、查封扣押、移送公安机关行政拘留以及涉嫌环境污染犯罪等五类案件**

**评查重点**：
- 参评人员对照评查细则
- 重点围绕执法主体是否适格、违法事实是否清楚、证据是否充分、法律适用是否正确、执法程序是否合法、文书制作是否规范等核心要素进行深入研讨
- 逐卷填写评审意见表
- 对典型问题和常见错误集中剖析交流
- 达到了纠偏补弱、规范引领的效果

**评查启示**：
1. 分轮次推进确保全覆盖且不影响日常工作
2. 五类案件全面评查无遗漏
3. 外部法律顾问参与提升专业性
4. 100余份案卷的大样本量确保统计意义


## 二、案卷评查技术革新

### 向"智能研判"进化：AI技术赋能案卷评查

**技术提供方**：法意科技（20年法律文本智能算法积淀）

**核心技术**：
1. **AI智能解析**：自动解析案卷文书结构
2. **规则引擎自动比对**：将案卷内容与评查细则自动比对
3. **前沿技术驱动**：实现评查效率的指数级跃升

**变革效果**：
1. **打破传统评查的经验依赖**：不再依赖评查人员个人经验
2. **构建统一、精准的标准化评查体系**：评查标准完全一致
3. **摆脱人力束缚**：自动化评查，减少人工成本
4. **推动从"人工核对"到"智能研判"的转型**：评查模式根本性变革
5. **为执法规范化、高效化注入持久动力**

**应用前景**：
- 可大幅缩短评查周期
- 提高评查准确率和一致性
- 实现评查全过程留痕可追溯
- 为评查大数据分析提供基础

**评查启示**：
1. 科技创新是提升评查质效的重要方向
2. AI技术可解决传统评查标准不一、效率低下的问题
3. 智能化评查是未来发展趋势


## 三、案卷评查共性经验总结

### （一）评查组织模式对比

| 模式 | 代表地区 | 核心特点 | 优势 |
|------|----------|----------|------|
| 显微式解剖 | 贺州市 | 7个方面逐卷逐页"过筛子" | 细致入微，问题找得准 |
| 两轮交叉+集中现场 | 赤峰市 | 两轮交叉互评+一轮集中现场 | 评查尺度统一，疑难问题集体研讨 |
| 一案双评+交叉互评 | 新余市 | 一案双评，统一复核 | 评查质量高，标准统一 |
| 四轮次全覆盖 | 济南市 | 分四轮次推进，实现全市全覆盖 | 不影响日常工作，全覆盖无遗漏 |
| 智能研判 | 科技赋能 | AI智能解析+规则引擎自动比对 | 效率高，标准统一，摆脱经验依赖 |

### （二）评查关键环节

根据各地评查实践，案卷评查应重点关注以下环节：

1. **执法主体**：是否适格
2. **事实认定**：违法事实是否清楚
3. **证据链条**：证据是否充分、确凿
4. **法律适用**：适用法律是否正确
5. **程序合规**：执法程序是否合法
6. **自由裁量**：裁量是否合理
7. **文书制作**：文书是否规范
8. **案卷归档**：归档是否规范

### （三）评查结果运用

1. **立即整改**：第一时间梳理汇总问题，制定针对性整改措施
2. **闭环管理**：推动各类问题整改落地落细、闭环清零
3. **效果跟踪**：跟踪整改效果，验证同类问题发生率是否下降
4. **纳入考核**：将评查结果纳入年度执法考核
5. **持续培训**：通过评训结合，推广优秀案卷示范经验
6. **岗位练兵**：持续开展业务培训和岗位练兵活动


## 四、学习总结

本次学习主要获取了2026年各地生态环境行政执法案卷评查的最新实践经验，涵盖了多种创新评查模式和组织形式。这些实践经验对于提升案卷评查质效、规范行政执法行为具有重要的借鉴意义。

**核心启示**：
1. 评查模式不断创新，从单一评查向综合评查、智能评查发展
2. 评查组织更加规范，交叉互评、专家点评、统一复核成为标配
3. 评查结果运用更加深入，整改闭环、纳入考核、持续培训形成合力
4. 科技赋能评查工作，AI技术开始应用于案卷评查领域

**应用建议**：
1. 在案卷评查工作中借鉴"显微式解剖"的细致态度
2. 推广"两轮交叉+集中现场"的评查模式
3. 建立评查专家库，引入专家点评机制
4. 强化评查结果运用，建立整改闭环机制
