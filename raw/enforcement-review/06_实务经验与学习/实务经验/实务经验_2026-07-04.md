## 一、评查新模式学习

### 1. 交叉评查模式

#### 贺州市"县区交叉互评 + 市局法制骨干复核"模式

**操作流程**:
1. **第一阶段**: 县区之间交叉互评
   - 避免人情干扰
   - 提高评查客观性
   - 促进县区之间交流学习

2. **第二阶段**: 市局法制骨干复核
   - 统一评查标准
   - 纠正评查偏差
   - 确保评查质量

**优点**:
- ✅ 避免"面子工程"，敢于较真碰硬
- ✅ 统一评查尺度，减少主观偏差
- ✅ 相互学习借鉴，提高评查能力

**评查要点**:
- 交叉评查人员安排要合理（避免利害关系）
- 评查标准要统一（集中培训、统一尺度）
- 复核机制要健全（市局复核、争议裁决）

#### 济南市"分轮次推进、全市全覆盖"模式

**操作流程**:
1. **分四轮次推进**: 每轮2-4个区县分局
2. **合理调配**: 根据各区县案卷数量合理调配
3. **全面覆盖**: 确保所有案卷都得到评查

**优点**:
- ✅ 分散评查压力，保证评查质量
- ✅ 灵活调配资源，提高效率
- ✅ 确保全覆盖，不漏一案

**评查要点**:
- 评查轮次安排要科学
- 评查人员分配要合理
- 评查进度要统筹

### 2. "显微式解剖"评查方法

#### 贺州市"显微式解剖"评查法

**评查深度**:
- 不只是看文书格式，更深入审查实质内容
- 不只是看程序合法性，更关注证据充分性
- 不只是看法律适用，更评估处罚适当性

**评查重点**:
1. **执法程序**:
   - 立案程序是否规范
   - 调查取证是否合法
   - 审查决定是否符合要求
   - 送达执行是否到位

2. **事实认定**:
   - 违法事实是否清楚
   - 证据材料是否充分
   - 证据链是否闭环
   - 事实认定是否准确

3. **法律适用**:
   - 法律条款引用是否准确
   - 处罚依据是否充分
   - 裁量幅度是否适当
   - 法律更新是否及时

**评查方法**:
- 逐页审阅案卷材料
- 逐项核对评查细则
- 深入分析问题根源
- 提出具体改进建议

## 二、新型违法案件评查经验

### 1. 技术性违法案件评查

#### 机动车排放造假案件评查要点

**案件特点**:
- 违法手段技术性强（OBD篡改、传感器造假）
- 证据固定需要专业技术
- 法律适用需要准确理解技术标准

**评查重点**:

1. **证据审查**:
   - [ ] 现场检查笔录是否详细记载违法部位
   - [ ] 现场照片是否清晰显示改装部位
   - [ ] 技术说明是否清楚解释危害机理
   - [ ] 证据链是否闭环（改装事实 → 危害后果）

2. **法律适用审查**:
   - [ ] 法律条款引用是否准确（大气污染防治法第55条第3款）
   - [ ] 处罚依据是否充分（大气污染防治法第112条第3款）
   - [ ] 处罚幅度是否符合规定（5000元罚款）

3. **程序审查**:
   - [ ] 现场检查程序是否规范
   - [ ] 询问笔录是否齐全
   - [ ] 权利告知是否到位
   - [ ] 送达程序是否合法

**评查风险提示**:
- ⚠️ 证据不足以证明技术改装事实 = 事实认定不清
- ⚠️ 法律适用错误（引用废止法律）= 一票否决
- ⚠️ 处罚幅度不当 = 规范性扣分

### 2. 监测数据造假案件评查

#### 第三方监测机构数据造假案件评查要点

**案件特点**:
- 违法手段隐蔽（软件编辑、数据篡改）
- 取证难度大（需要技术手段）
- 危害面广（影响多个企业）

**评查重点**:

1. **证据审查**:
   - [ ] 监测报告与原始记录是否比对
   - [ ] 电子证据是否及时固定（封存电脑、拷贝数据）
   - [ ] 软件造假是否查实（可编辑文档、修改痕迹）
   - [ ] 证据链是否闭环（造假手段 → 虚假报告）

2. **双罚制审查**:
   - [ ] 是否同时处罚公司
   - [ ] 是否同时处罚责任人
   - [ ] 责任人认定是否准确
   - [ ] 罚款金额是否符合裁量基准

3. **法律适用审查**:
   - [ ] 法律条款引用是否准确（地方性法规或部门规章）
   - [ ] 处罚依据是否充分
   - [ ] 裁量基准适用是否合理

**评查风险提示**:
- ⚠️ 电子证据未及时固定 = 证据可能灭失
- ⚠️ 未落实双罚制 = 处罚不全面
- ⚠️ 裁量基准适用不当 = 处罚不当

### 3. 重金属超标案件评查

#### 含重金属废水排放案件评查要点

**案件特点**:
- 一类污染物必须在车间排放口采样
- 超标3倍以上达到刑事立案标准
- 私设暗管逃避监管处罚加重

**评查重点**:

1. **采样程序审查**:
   - [ ] 一类污染物是否在车间排放口采样（评查细则强制要求）
   - [ ] 采样点位是否准确
   - [ ] 采样记录是否完整
   - [ ] 封存程序是否规范

2. **监测报告审查**:
   - [ ] 监测报告是否标注采样点位
   - [ ] 监测报告是否标注采样时间
   - [ ] 监测报告是否标注监测方法
   - [ ] 监测结果是否准确（超标倍数计算）

3. **移送标准审查**:
   - [ ] 是否达到刑事立案标准（重金属≥3倍）
   - [ ] 移送依据是否充分
   - [ ] 移送文书是否齐全
   - [ ] 移送时间是否及时

**评查风险提示**:
- ⚠️ 一类污染物未在车间排放口采样 = 程序违法（一票否决）
- ⚠️ 监测报告不规范 = 证据不足
- ⚠️ 达到移送标准未移送 = 履职不到位


## 三、程序法适用评查经验

### 1. 行政复议权利告知审查

#### 2026年7月1日起施行新《行政复议法实施条例》

**评查重点**:

1. **复议权利告知完整性**:
   - [ ] 是否告知复议权利
   - [ ] 是否告知复议机关（本级政府或上级主管部门）
   - [ ] 是否告知复议申请期限（60日）
   - [ ] 是否告知诉讼权利（6个月）

2. **复议机关告知准确性**:
   - [ ] 复议机关名称是否正确
   - [ ] 复议机关地址是否准确
   - [ ] 联系方式是否提供

3. **期限计算准确性**:
   - [ ] 复议申请期限（60日）告知是否正确
   - [ ] 诉讼期限（6个月）告知是否正确
   - [ ] 期限计算是否从送达之日起计算

**评查风险提示**:
- ⚠️ 未告知复议权利 = 程序违法
- ⚠️ 复议机关告知错误 = 程序瑕疵
- ⚠️ 期限告知错误 = 程序瑕疵

### 2. 起诉期限审查

#### 行政诉讼法起诉期限规定

**评查重点**:

1. **起诉期限告知**:
   - [ ] 是否告知起诉期限（6个月）
   - [ ] 期限计算是否准确
   - [ ] 起诉法院是否告知

2. **期限计算**:
   - [ ] 复议决定送达之日起6个月内
   - [ ] 直接起诉：知道或应当知道行政行为之日起6个月内
   - [ ] 最长保护期限：5年（不动产案件20年）

**评查风险提示**:
- ⚠️ 未告知起诉期限 = 程序瑕疵
- ⚠️ 期限告知错误 = 程序瑕疵

2. **了解案情背景**:
   - 阅读处罚决定书
   - 了解违法事实
   - 掌握案件特点

3. **准备评查工具**:
   - 评查细则文本
   - 法律文书格式范本
   - 法律法规汇编

### 2. 评查中的工作方法

1. **逐卷审阅**:
   - 从头到尾仔细阅读
   - 不放过任何细节
   - 做好阅评记录

2. **逐项核对**:
   - 对照评查细则逐项核对
   - 有问题的地方标记出来
   - 扣分要有依据

3. **深入思考**:
   - 不只是看表面问题
   - 深入分析问题根源
   - 提出改进建议

### 3. 评查后的工作总结

1. **评分要客观**:
   - 评分要有依据
   - 扣分要说明理由
   - 避免主观臆断

2. **建议要具体**:
   - 改进建议要可操作
   - 指出具体修改方向
   - 提供法条依据

3. **经验要总结**:
   - 总结评查经验
