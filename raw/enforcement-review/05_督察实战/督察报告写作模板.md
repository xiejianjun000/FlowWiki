# 督察报告写作模板

> 省级生态环境保护督察反馈意见/督察报告的标准结构模板。

## 使用说明
- 本模板适用于省级督察反馈意见的撰写
- `[方括号]` 内容需替换为实际信息
- 每个问题必须有具体点位、时间、数据支撑
- 引用 [[政府借力与专报撰写]] 的写作技巧

## 反馈意见模板

```
──────────────────────────────────────────
XX省第X生态环境保护督察组
向XX市（州）反馈督察意见

（XX年XX月XX日）
──────────────────────────────────────────

为深入贯彻落实习近平生态文明思想，根据《XX省生态环境保护督察工作实施办法》，XX年XX月XX日至XX月XX日，XX省第X生态环境保护督察组对XX市（州）开展了生态环境保护督察。督察组通过听取汇报、调阅资料、个别谈话、现场核查、受理群众举报等方式，形成督察反馈意见。经省委、省政府批准，现将督察意见反馈如下：

一、总体评价

[1-2段]
XX市（州）党委、政府积极践行习近平生态文明思想，在[具体工作]方面取得一定成效。但督察发现，XX市（州）在[具体领域]仍存在突出短板，[……]。

主要问题：

（一）贯彻落实习近平生态文明思想有差距

1. 思想认识不到位。……
2. 责任落实有偏差。……
3. 考核问责流于形式。……

（二）[领域A]问题突出

1. [具体问题1]：XX企业/园区/点位，[违规行为]，[持续时间]。[数据支撑]
2. [具体问题2]：……
3. [具体问题3]：……

（三）[领域B]整改不到位

1. 上一轮督察反馈的[具体问题]，至今未完成整改。
2. XX信访件，群众多次重复举报，问题仍未解决。
3. ……

（四）[领域C]环境风险隐患

1. ……
2. ……

意见和建议：

（一）进一步提高政治站位。……
（二）限期整改交办问题。对督察组交办的问题，30日内报送整改方案，6个月内报送整改落实情况。
（三）严肃责任追究。对履职不力、失职失责的，依纪依法追责问责。
（四）举一反三建立长效机制。……

──────────────────────────────────────────
```

## 问题描述"五要素"规则

每个问题必须包含：

| 要素 | 示例 |
|------|------|
| **主体** | XX市XX县XX企业 |
| **行为** | 未经环评审批擅自建设XX生产线 |
| **时间** | 2019年至督察进驻期间 |
| **后果** | 产生XX吨危险废物，违规堆放未采取"三防"措施 |
| **数据** | 周边地下水XX指标超标XX倍 |

❌ 不合格写法："XX市对环保工作重视不够"
✅ 合格写法："XX市2019-2025年间未经环评审批违规上马12个高污染项目，涉及XX、XX等行业"

## 问题定级标准

| 等级 | 标准 | 处理方式 |
|------|------|---------|
| 🔴 重大问题 | 触碰法律法规红线、造成严重环境损害、敷衍整改/虚假整改 | 移交追责、限期整改、公开曝光 |
| 🟠 重点问题 | 区域性/行业性普遍问题、长期未解决的突出问题 | 限期整改、重点督办 |
| 🟡 一般问题 | 个别点位/企业的具体问题 | 立行立改、纳入整改清单 |

## 常见写作错误

1. **"万能句"堆砌** — "认识不到位""力度不够""重视不足"三连击，没有具体事实。→ 每段必须有具体案例
2. **数据空洞** — 只说"严重超标"，不说超标多少倍、什么指标。→ 定量优先
3. **责任模糊** — 不说哪个部门/哪级政府的问题。→ 落实到具体责任主体
4. **建议虚化** — "切实加强""进一步重视"等无约束力的建议。→ 建议必须有时间节点和可考核标准
