# 督察问题分类与整改跟踪

> 督察反馈后的问题全生命周期管理：分类 → 交办 → 调度 → 验收 → 销号 → 回头看。

## 何时读我
- 督察反馈意见已签发，需要组织整改
- 调度整改进展，防止"材料整改"
- 验收销号，判断问题是否真正解决

---

## 一、问题分类体系

### 按领域分类

| 大类 | 子类 | 典型问题 |
|------|------|---------|
| 政治站位 | 思想认识、责任落实、考核问责 | 党委常委会全年未研究环保工作 |
| 产业结构 | 违规审批、"两高"项目、落后产能淘汰 | 违规备案限制类项目 |
| 污染防治 | 水/气/土/固废/噪声 | 污水处理厂长期超标排放 |
| 生态保护 | 自然保护区、生态红线、矿山修复 | 保护区内违规建设 |
| 环境基础设施 | 污水管网、垃圾处理、固废处置 | 管网空白区、雨污不分 |
| 执法监管 | 执法力度、监测能力、应急能力 | "零处罚"县、监测数据造假 |
| 群众信访 | 重复举报、久拖不决 | 同一问题3次以上举报未解决 |

### 按严重程度分级

| 等级 | 定义 | 验收标准 | 例 |
|------|------|---------|-----|
| A级（重大问题） | 触碰法律红线、严重环境损害 | 措施完全到位 + 现场核验 + 追责到位 | 暗管偷排危废 |
| B级（重点问题） | 区域性/行业性普遍问题 | 制度建立 + 整改率>90% + 抽查合格 | 园区管网不配套 |
| C级（一般问题） | 个别点位问题 | 立行立改 + 照片/数据佐证 | 企业台账不规范 |

---

## 二、整改方案审核清单

收到被督察方整改方案时，逐项审核：

- [ ] 是否"一问题一方案"？（不可多个问题合并一个方案）
- [ ] 整改措施是否具体可考核？（禁止"加强学习""提高认识"等虚化措施）
- [ ] 是否明确责任单位+责任人？（落实到具体部门和姓名）
- [ ] 是否明确完成时限？（精确到月，不可"长期坚持"）
- [ ] 是否建立长效机制？（举一反三，防止同类问题复发）
- [ ] 资金、技术保障是否落实？（不能"无米之炊"）

---

## 三、整改调度节点

| 节点 | 时间 | 动作 |
|------|------|------|
| T+30日 | 反馈后1个月 | 审核整改方案，不合格退回重报 |
| T+90日 | 反馈后3个月 | 中期调度：调取进展报告+现场抽查不少于30%的问题 |
| T+180日 | 反馈后6个月 | 整改落实情况报告审核+全面验收 |
| 长期 | 不定期 | "回头看"：防止反弹，重点关注A级问题和重复举报问题 |

---

## 四、验收销号标准

### A级问题验收
- [ ] 违法行为已停止
- [ ] 环境损害已修复（或达修复方案阶段目标）
- [ ] 责任追究已完成
- [ ] 长效机制已建立
- [ ] 现场核验通过

### B级问题验收
- [ ] 制度文件已出台
- [ ] 整改率达90%以上
- [ ] 抽查点位合格率>80%
- [ ] 同类问题发生率下降

### C级问题验收
- [ ] 立行立改完成
- [ ] 照片/数据佐证材料齐全

## 五、常见整改陷阱与应对

| 陷阱 | 表现 | 应对 |
|------|------|------|
| **材料整改** | 整改报告写得好，现场纹丝不动 | 验收必须现场核验，不可仅凭报告 |
| **选择性整改** | 容易的改了，难的拖延 | 重点调度A级问题，建立"硬骨头"专项台账 |
| **搭车整改** | 借整改之名"一刀切"关停合法企业 | 区分合法企业与违法企业，保护合法经营 |
| **甩锅整改** | 把问题归结为"历史遗留""上级原因" | 明确现职责任，不管历史成因，只管现在谁负责解决 |
| **反复整改** | 同一问题多次列入整改仍不解决 | 升级处理：重点督办→约谈→移交追责 |
