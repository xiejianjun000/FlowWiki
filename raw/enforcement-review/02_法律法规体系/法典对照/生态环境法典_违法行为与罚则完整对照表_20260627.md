# 中华人民共和国生态环境法典——违法行为与罚则完整对照表

**研究日期**：2026年6月27日  
**施行日期**：2026年8月15日  
**法律依据**：《中华人民共和国生态环境法典》第五编（第五编：法律责任和附则）  
**条文范围**：第1052条—第1238条  
**说明**：本表依据法典原文逐条梳理，共覆盖14节违法行为类型，是目前最完整的违法行为与罚则对照表。

## 【通则篇】法律责任通则（第1052条—第1084条）

### 一、基本原则（第1052—1058条）

| 法条 | 核心内容 | 实务要点 |
|------|---------|---------|
| 第1052条 | 任何单位和个人有污染环境、破坏生态和违反绿色低碳义务的行为，应当严格依法追究法律责任 | 确立了"损害担责"原则 |
| 第1053条 | 污染环境/破坏生态行为：行政责任适用**过错推定**（有证据证明无过错的除外）；民事责任适用**无过错责任** | 举证责任：企业须举证证明无过错，否则承担行政责任 |
| 第1054条 | 追责期限：**造成危害后果的5年**，其他违法行为2年（均从行为发生之日或行为终了之日起计算）；生态环境民事诉讼时效：**5年** | ⚠️ 5年期限为法典新规，旧法为一般2年；5年时效从知道或应当知道损害及责任人起算 |
| 第1055条 | 同一违法行为违反多个法律规范的，**按罚款数额高的规定处罚** | 吸收"一事不再罚"原则中的罚款竞合规则 |
| 第1056条 | 从重：恶意损害、拒不改正、严重后果、多次被罚；从轻/减轻：主动消除危害、修复措施、缴纳赔偿金；**不予处罚**：轻微+及时改正无后果；初次违法+轻微+及时改正 | ⚠️ 法典统一规定，各地裁量基准需据此修订 |
| 第1057条 | 从旧兼从轻：适用违法行为发生时的规定，但新规处罚较轻或不为违法的适用新规 | 与《行政处罚法》第37条一致 |
| 第1058条 | 民事责任优先；行政拘留可折抵刑期；行政罚款可折抵罚金 | 民行刑衔接的核心条款 |

### 二、按日连续处罚（第1060条）

| 适用条件 | 程序要点 | 罚款计算 |
|---------|---------|---------|
| ①违法排放/倾倒/处置污染物<br>②受到罚款处罚<br>③被责令改正<br>④拒不改正或拒绝/阻挠复查 | 复查→发现继续违法→自责令改正次日按日连续 | 原罚款数额（按运行成本/直接损失/违法所得等因素在法定幅度内确定） |

> **地方立法授权**：地方性法规可根据实际需要，增加按日连续处罚的违法种类。

### 三、责任主体（第1061—1071条）

| 法条 | 主体 | 责任形式 |
|------|------|---------|
| 第1061条 | 地方人民政府（未依法履行职责） | 对负责人依法给予处分或处理，并可向社会公开 |
| 第1062条 | 地方各级人民政府及负有监管职责的部门/机构工作人员 | 记过、记大过、降级；造成严重后果的撤职或开除（包庇违法、未依法查处、篡改监测数据等10种情形） |
| 第1063条 | 规划编制/审批机关及生态环境主管部门 | 依法给予处分（环评失实、违法收取费用等） |
| 第1064条 | 企业事业单位和其他生产经营者及责任人 | 依法承担法律责任（含法定代表人/主要负责人/直接责任人） |
| 第1065条 | 造成污染/破坏者 | 消除污染、修复生态；无法修复的实施替代性治理；违法行为人变更的由承继者负责 |
| 第1067条 | 生态环境技术服务机构（受委托方） | 与委托人承担**连带责任** |
| 第1068条 | 委托倾倒/运输/处置方；境外固废进口承运人 | 与受托人/进口者承担**连带责任** |
| 第1069条 | 公职人员属于责任人员的 | 依法给予处分 |

### 四、责任追究程序（第1072—1084条）

| 法条 | 机制 | 实务要点 |
|------|------|---------|
| 第1072条 | 上级政府可对下级直接作出处罚决定 | 层级监督 |
| 第1073条 | 生态环境损害赔偿磋商（省级政府/指定部门与责任人）→磋商不成可起诉；检察机关可提起诉讼 | ⚠️ 民事赔偿优先 |
| 第1074条 | 海洋污染损害赔偿：行使海洋监管权的部门→检察院可起诉 | 海洋专属 |
| 第1075条 | 损害社会公共利益：检察院或符合条件的社会组织→公益诉讼 | 环境公益诉讼 |
| 第1076条 | 检察院发现政府/监管部门违法→提起诉讼 | 行政公益诉讼 |
| 第1079条 | 当事人可向法院申请**禁止令**（立即停止污染/破坏行为） | 行为保全 |
| 第1080条 | 环境污染/破坏侵权：行为人承担**举证责任**（法定免责/减责情形及因果关系不存在） | 因果关系举证责任倒置 |
| 第1083条 | 涉嫌犯罪→移送侦查机关 | 行政执法与刑事司法衔接 |
