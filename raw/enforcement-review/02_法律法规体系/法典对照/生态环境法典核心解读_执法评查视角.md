# 《中华人民共和国生态环境法典》核心解读——执法评查视角

## 三、执法评查重点变化

### （一）行政处罚时效重大调整（第1054条）

| 违法类型 | 追究时效 | 原法规定 |
|---------|---------|---------|
| 造成环境污染、生态破坏等危害后果的违法行为 | **5年** | 2年（原《行政处罚法》） |
| 其他违法行为 | 2年 | 2年 |

**评查影响**：对超标排放、非法排污等后果型违法行为的处罚追究期从2年延长至5年，评查时需注意案卷中违法行为发现时间是否在追究时效内。

### （二）民事公益诉讼时效（第1054条第三款）

- 提起生态环境民事诉讼的诉讼时效期间为**5年**（比一般民事诉讼3年更长）
- 自当事人知道或应当知道受到损害以及责任人之日起计算

### （三）按日连续处罚纳入法典（第1056条）

- 按日连续处罚制度正式写入法典
- 适用于被责令改正拒不改正的情形
- **评查要点**：复查期限、计罚方式、处罚决定书的规范性

### （四）查封扣押制度（第52条、第1057条）

- 负有监管职责的部门可依法查封、扣押有关场所、船舶、设施、设备、工具、物品
- 适用条件：违法造成或可能造成严重后果，或证据可能灭失、被隐匿

### （五）派出机构执法权限明确（第53条）

- **设区的市级人民政府生态环境主管部门设立的派出机构**，可以以自己的名义实施：
  - 现场检查
  - 查封、扣押
  - 代履行
  - **行政处罚**

**评查影响**：分局/大队以自己名义作出处罚决定的主体资格问题，法典已明确授权。

### （六）排污许可制度重大整合（第174-186条）

- 排污许可制成为**固定污染源监督管理的核心制度**
- 新增电磁辐射设施纳入排污许可管理
- 排污许可证有效期5年
- 自行监测记录和管理台账保存期限**不少于5年**

### （七）逃避监管排放方式明确列举（第164条）

禁止通过以下方式排放污染物：
1. 暗管
2. 渗井、渗坑
3. 灌注
4. 裂隙、溶洞
5. 篡改或伪造监测数据
6. 以逃避现场检查为目的的临时停产
7. **非紧急情况下开启应急排放通道**（新增）
8. 不正常运行污染防治设施

**评查影响**：比原《环境保护法》第42条和《水污染防治法》第39条更加细化的列举，评查中认定"逃避监管方式排放"的依据更充分。

### （八）生态环境监测数据造假（第80条）

明确禁止的弄虚作假方式：
1. 干扰采样
2. 调换样品
3. 改变监测条件
4. 虚假监测
5. 篡改或伪造记录
6. 指使弄虚作假

同时禁止干扰、破坏监测设施、设备。

### （九）法律责任分则体系（第五编第二章）

14节分则对应不同违法类型，处罚体系更加系统化：

| 节 | 违法类型 | 典型罚则 |
|---|---------|---------|
| 第一节 | 违反监测管理规定 | 弄虚作假：10-200万元 |
| 第二节 | 违反环评管理规定 | 未批先建：项目投资额1%-5% |
| 第三节 | 违反排污许可管理规定 | 无证排污：20-100万元 |
| 第四节 | 违反大气污染防治规定 | 逃避监管：10-100万元 |
| 第五节 | 违反水污染防治规定 | 逃避监管：10-100万元 |
| 第六节 | 违反海洋污染防治规定 | - |
| 第七节 | 违反土壤污染防治规定 | - |
| 第八节 | 违反固废污染防治规定 | - |
| 第九节 | 违反噪声污染防治规定 | - |
| 第十节 | 违反放射性污染防治规定 | - |
| 第十一节 | 违反化学物质/电磁/光污染防治规定 | - |
| 第十二节 | 违反生态保护管理规定 | - |
| 第十三节 | 违反绿色低碳发展管理规定 | 碳排放未清缴：100-500万元 |
| 第十四节 | 其他违法行为 | 事故罚款：30万-2000万元 |

### （十）环境污染事故分级处罚（第1236条）

| 事故等级 | 罚款幅度 |
|---------|---------|
| 一般 | 30万-100万元 |
| 较大 | 100万-200万元 |
| 重大 | 200万-1000万元 |
| 特别重大 | 1000万-2000万元 |
| 情节特别严重、影响特别恶劣 | 2-5倍 |


## 四、对案卷评查实务的影响

### 1. 法律适用更新

- 2026年8月15日后，案卷中引用的法律依据必须从法典条文，不再引用已废止的10部单行法
- **过渡期**：发生在法典施行前的行为，按"从旧兼从轻"原则处理

### 2. 评查细则需同步更新

- 处罚时效从2年→5年的变化，影响评查中对"是否超过追究时效"的判断
- 派出机构执法权限明确，不再以"超越职权"为由认定处罚主体不合法
- 逃避监管方式新增列举项，评查认定标准需调整

### 3. 新增评查关注点

- 排污许可与环评、监测等制度的衔接是否规范
- 按日连续处罚的程序合规性（责令改正→复查→计罚）
- 生态环境损害赔偿与行政处罚的衔接
- 碳排放配额清缴相关处罚的程序规范

### 4. 全国生态日

- 每年8月15日为全国生态日（第15条），与法典施行日重合
