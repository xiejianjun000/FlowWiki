## 案例一：破坏机动车车载排放诊断系统案

### 基本信息
- **案例编号**: 2026-07-04-001
- **案例类型**: 大气污染防治/机动车排放造假
- **违法主体**: 开封市XX运输有限公司
- **违法地点**: 河南省郑州市新郑市郭店镇玉泽物流园
- **查处机关**: 郑州市生态环境局
- **入选批次**: 郑州市2026年第四批生态环境执法典型案例

### 案情简介

#### 违法事实
2025年8月25日，郑州市生态环境局执法人员根据生态环境部监督帮扶要求，对位于新郑市郭店镇107国道东侧小李庄村东200米的玉泽物流园进行检查，发现：

1. **违法车辆**: 车牌号豫B006XX的重型柴油货车
2. **所有人**: 开封市XX运输有限公司
3. **违法手段**: 温度传感器加装垫高螺丝
4. **技术细节**: 垫高约5厘米
5. **危害机理**: 
   - 减少机动车尾气治理中尿素喷射量
   - 直接导致SCR系统（选择性催化还原系统）处理氮氧化物效率降低
   - 尾气排放浓度升高
   - 破坏了机动车车载排放诊断控制系统的温度测试单元的正常运行

#### 现场证据固定
- 违法车辆现场照片
- 温度传感器垫高部位特写照片
- 垫高螺丝尺寸测量记录（约5厘米）
- 车辆所有人资质证明材料

### 法律适用

#### 违法条款
**《中华人民共和国大气污染防治法》第五十五条第三款**:
> "禁止机动车所有人以临时更换机动车污染控制装置等弄虚作假的方式通过机动车排放检验。禁止机动车维修单位提供该类维修服务。禁止破坏机动车车载排放诊断系统。"

#### 处罚依据
**《中华人民共和国大气污染防治法》第一百一十二条第三款**:
> 违反本法规定，破坏机动车车载排放诊断系统的，由县级以上人民政府环境保护主管部门责令改正，处五千元的罚款。

#### 处罚决定
- **处罚机关**: 郑州市生态环境局
- **处罚日期**: 2025年9月28日
- **处罚内容**: 罚款人民币5000元

### 案件评析

#### 新型违法特点
1. **技术性强**: 通过对传感器进行物理改装，逃避监管
2. **隐蔽性高**: 外观不易察觉，需要专业检查才能发现
3. **危害性大**: 直接导致尾气排放超标，污染大气环境
4. **代表性**: 反映当前机动车排放造假的新趋势

#### 取证要点
1. **现场检查**: 
   - 检查温度传感器安装位置
   - 测量垫高螺丝尺寸
   - 拍照固定证据
   
2. **技术验证**:
   - 核实温度传感器工作原理
   - 分析垫高对温度检测的影响
   - 评估对SCR系统效率的影响

3. **证据链闭环**:
   - 现场照片 → 证明垫高事实
   - 技术说明 → 证明危害机理
   - 车辆信息 → 证明违法主体

**规范性审查**:

## 案例二：第三方监测机构伪造监测数据案

### 基本信息
- **案例编号**: 2026-07-04-002
- **案例类型**: 环境监测数据造假/第三方机构违法
- **违法主体**: 广西某检测技术有限公司
- **违法地点**: 广西壮族自治区钦州市
- **查处机关**: 钦州市生态环境局
- **入选批次**: 自治区生态环境厅2026年第一批生态环境执法典型案例

### 案情简介

#### 违法事实
2025年10月，钦州市生态环境局执法人员在开展全国远程大气监督帮扶问题线索核查时，发现辖区部分企业自行监测报告存在多处异常：

1. **初步发现**:
   - 所附仪器原始记录小票字体与标准制式小票不符
   - 不同文件号的采样小票中，工况体积和标况体积数据高度雷同

2. **现场核查**（2025年10月27日）:
   - 对报告出具单位广西某检测技术有限公司进行现场检查
   - 查实该公司烟气综合测试仪原始操作记录与监测报告附载小票的数据、文件号互不对应
   - 采样时长存在明显造假

3. **具体违法手段**:
   - **数据篡改**: 烟气综合测试仪原始操作记录显示实际采样运行仅10秒，但监测报告附载小票同一时间段却标注为采样运行20分钟
   - **软件造假**: 在该公司办公电脑中查获可人为编辑篡改的烟尘测试仪采样报表文档
   - **虚构数据**: 违反监测规范编造出具虚假监测报告

#### 现场证据固定
- 监测报告与原始记录比对材料
- 电脑中查获的 editable 采样报表文档
- 采样时长对比分析（10秒 vs 20分钟）
- 数据雷同分析报告

### 法律适用

#### 违法条款
**《广西壮族自治区环境保护条例》第十七条第一款**:
> 环境监测机构应当按照国家有关规定，从事环境监测活动，不得弄虚作假、隐瞒、伪造、篡改环境监测数据。

#### 处罚依据
**《广西壮族自治区环境保护条例》第五十七条第一款**:
> 环境监测机构弄虚作假的，由县级以上地方人民政府环境保护主管部门责令改正，处五万元以上十万元以下的罚款；对直接负责的主管人员和其他直接责任人员，由县级以上地方人民政府环境保护主管部门给予警告，处二万元以上五万元以下的罚款。

#### 处罚决定
- **处罚机关**: 钦州市生态环境局
- **处罚日期**: 2026年3月16日
- **处罚内容**:
  - 对广西某检测技术有限公司: 罚款5.2062万元
  - 对相关负责人1: 罚款2.1012万元
  - 对相关负责人2: 罚款2.0675万元
- **裁量基准**: 《广西壮族自治区生态环境行政处罚裁量规则(2023版)》

### 案件评析

#### 案件特点
1. **隐蔽性强**: 通过软件编辑篡改数据，外观难以发现
2. **危害面广**: 监测数据造假影响多个企业，扰乱市场秩序
3. **技术含量高**: 需要懂监测技术、懂软件操作才能发现和取证
4. **双罚制**: 既罚公司，又罚责任人

#### 取证要点
1. **线索发现**:
   - 关注监测报告异常（字体、数据雷同）
   - 比对原始记录与报告数据
   
2. **现场检查**:
   - 检查监测仪器设备原始记录
   - 检查办公电脑存储文档
   - 比对采样时长真实性
   
3. **证据固定**:
   - 打印电脑中 editable 文档
   - 截图保存软件界面
   - 制作比对分析表

**规范性审查**:
- [x] 线索核查程序规范
- [x] 现场检查笔录详细

**扣分风险点**:

### 启示与建议

## 案例三：私设暗管排放含重金属废水案

### 基本信息
- **案例编号**: 2026-07-04-003
- **案例类型**: 水污染防治/逃避监管/重金属超标
- **违法主体**: 武汉某光电子有限公司
- **违法地点**: 湖北省武汉市黄陂区某工业园
- **查处机关**: 武汉市生态环境保护综合执法支队、武汉市生态环境局黄陂区分局
- **入选批次**: 湖北省2026年第二批生态环境执法典型案例（典型指导案例）

### 案情简介

#### 违法事实
2025年9月11日，根据省厅交办线索，武汉市生态环境保护综合执法支队会同武汉市生态环境局黄陂区分局对某工业园内非法从事表面处理加工的企业进行了突击检查，发现：

1. **企业基本情况**:
   - 企业名称: 武汉某光电子有限公司
   - 生产内容: 从事金属表面处理及热处理
   - 地点: 工业园2栋6楼（顶层）

2. **违法事实**:
   - **未批先建**: 未办理环评审批手续
   - **无证排污**: 未办理排污许可手续
   - **无治理设施**: 未建设污水处理设施
   - **私设暗管**: 加工车间生产废水未经处理，通过私设暗管的方式接入外墙楼顶雨水管，直接排放至市政雨水管网

3. **监测结果**（现场取样检测）:
   - **车间总排口排放污水**:
     - 六价铬浓度: 3.92 mg/L
     - 总铬浓度: 6.51 mg/L
     - 镍浓度: 1.52 mg/L
   - **超标倍数**（《电镀污染物排放标准》GB21900-2008 表2）:
     - 六价铬: 超标19.6倍（限值0.2 mg/L）
     - 总铬: 超标6.51倍（限值1.0 mg/L）
     - 镍: 超标3.04倍（限值0.5 mg/L）

#### 现场证据固定
- 企业生产车间照片
- 私设暗管走向示意图
- 暗管接入雨水管网照片
- 现场取样记录
- 监测报告（显示重金属超标）

### 法律适用

#### 违法条款
**《中华人民共和国水污染防治法》第三十九条**:
> "禁止利用渗井、渗坑、裂隙、溶洞，私设暗管，篡改、伪造监测数据，或者不正常运行水污染防治设施等逃避监管的方式排放水污染物。"

#### 移送依据
**《最高人民法院、最高人民检察院关于办理环境污染刑事案件适用法律若干问题的解释》**:
> 排放、倾倒、处置含铅、汞、镉、铬、砷、铊、锑的污染物，超过国家或者地方污染物排放标准3倍以上的，应当认定为"严重污染环境"，追究刑事责任。

#### 处理情况
- **行政违法认定**: 违反《水污染防治法》第39条
- **刑事移送**: 涉嫌污染环境罪
- **移送机关**: 公安机关
- **量化标准**: 六价铬超标19.6倍、总铬超标6.51倍、镍超标3.04倍，均超过3倍标准

### 案件评析

#### 案件特点
1. **多重违法**: 未批先建 + 无证排污 + 私设暗管 + 超标排放
2. **危害严重**: 含重金属废水直排，对环境造成严重污染
3. **隐蔽性强**: 暗管设置在楼顶雨水管，不易发现
4. **达到刑责**: 重金属超标3倍以上，追究刑事责任

#### 取证要点
1. **现场检查**:
   - 检查生产车间布局
   - 追踪废水流向
   - 发现暗管走向
   
2. **采样监测**:
   - **一类污染物**（六价铬、总铬、镍）必须在车间排放口采样
   - 规范采样程序
   - 同步采样、同步封存
   
3. **证据固定**:
   - 暗管照片、走向图
   - 采样记录、封存记录
   - 监测报告

#### 评查重点

**合法性审查**:
- [x] 执法主体适格
- [x] 采样程序合法（一类污染物车间排放口采样）
- [x] 监测结果准确（超标3倍以上）
- [x] 移送依据充分（达到刑事立案标准）

**规范性审查**:
- [x] 现场检查笔录详细
- [x] 采样记录完整
- [x] 监测报告规范
- [x] 移送文书齐全

**扣分风险点**:
- 需检查采样点位是否为一类污染物车间排放口（评查细则要求）
- 需检查监测报告是否标注采样点位、采样时间、监测方法

### 启示与建议

#### 对执法人员的建议
1. **重视一类污染物**: 必须在车间排放口采样
2. **追踪废水走向**: 不放弃任何暗管线索
3. **规范采样程序**: 采样点位、采样方法、封存程序都要规范
4. **及时移送公安**: 达到量化标准要及时移送

#### 对评查人员的建议
1. **审查采样点位**: 一类污染物是否在车间排放口采样
2. **审查监测报告**: 是否标注采样点位、方法、时间
3. **审查移送标准**: 是否达到刑事立案量化标准
4. **审查移送程序**: 移送文书是否齐全、移送时间是否及时
