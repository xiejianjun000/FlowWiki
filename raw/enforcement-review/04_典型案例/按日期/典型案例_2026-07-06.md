## 一、机动车排放检验造假（第三方机构）

### 案例1：甘肃兰州某机动车检测公司使用OBD作弊器出具虚假报告案
- **案情**：2025年12月，该公司使用OBD作弊器辅助**54辆机动车**通过排放检测，违规出具合格检验报告。
- **处理**：兰州市生态环境局依据《大气污染防治法》对企业**罚款、没收违法所得**，并将线索**移交市场监管部门**后续处理。
- **评查启示**：
  - 第三方机动车检测造假：**生态环境部门处罚 + 移送市场监管**双线处理模式。
  - 证据要点：OBD作弊器实物/使用记录、检测过程视频、54辆车清单与对应报告。
  - 移送市场监管的衔接文书须入卷，体现"行刑/行管衔接"闭环。

### 案例2：宁夏银川某机动车辆检测公司虚假检测报告生态环境损害赔偿案
- **案情**：涉嫌提供虚假证明文件、虚假检测报告造成大气污染，纳入生态环境损害赔偿。
- **评查启示**：检测造假除行政处罚外，可同步启动生态环境损害赔偿；案卷应含赔偿磋商/认定文书。

## 二、自动监测数据造假（刑事追责新类型）

### 案例3：湖北襄阳某重点排污单位VOCs自动监测数据造假刑事立案案【省内首例】
- **案情**：企业篡改挥发性有机物（VOCs）自动监测数据、逃避监管排污。
- **意义**：系**2026年新版环境污染刑事案件司法解释（法释〔2026〕4号）实施后，湖北省首例VOCs监测数据弄虚作假刑事追责案件**，填补省内此类案件刑事查处空白。
- **评查启示**：
  - **移送公安类案卷新标杆**：VOCs自动监测数据造假已明确纳入刑事追责，移送量化标准适用法释〔2026〕4号。
  - 证据链：自动监测平台原始数据、运维日志、篡改痕迹（参数修改/干扰采样）、比对监测报告、嫌疑人供述。
  - 此类案卷评查须重点核查"移送标准达成 + 证据链闭环 + 移送文书规范"。


## 三、固体废物 / 危险废物违法

### 案例4：宁夏吴忠同心县王某违法倾倒染料生产废液损害土壤刑事附带民事公益诉讼案
- **案情**：违法倾倒染料生产废液损害土壤，提起刑事附带民事公益诉讼。
- **评查启示**：危废/废液倾倒类，注意"刑事 + 民事公益诉讼 + 生态环境损害赔偿"多程序并行；案卷若由生态环境部门移送，须含移送公安文书、鉴定报告（危废属性/数量）、损害评估。

### 案例5：新疆昌吉呼图壁县某科技公司员工陈某擅自倾倒堆放危险废物案（既往典型，模式可鉴）
- **要点**：现场拍照录像锁定证据 → 查封决定书+清单防证据灭失 → 危废鉴定（HW08，49.5吨>3吨）→ 移送公安。
- **评查启示**：危废总量超3吨即触刑，评查移送类案卷须核对"称重/鉴定+移送"双要件齐全。


## 四、不正常运行污染防治设施

### 案例6：湖北鄂州某工贸有限公司不正常运行大气污染防治设施案
- **案情**：2024年8月9日检查，氧化车间酸洗作业期间，配套两台酸雾废气碱液喷淋塔均未运行，皮带静止、喷淋未开启。
- **评查启示**：
  - "不正常运行"认定需现场笔录固定"设施未运行"客观状态 + 生产正在进行（同时段性）。
  - 证据：现场照片/视频（皮带静止、无喷淋）、笔录、可能的用电/台账佐证。
  - 宁夏石嘴山某司同类"不正常运行防治设施造成大气污染生态环境损害赔偿案"可作对照。


## 五、噪声污染（群众举报 + 第三方检测 + 奖励）

### 案例7：湖北十堰某司噪声超标案（群众举报奖励 + 化解舆情）
- **案情**：2023年10月，市民12345举报生产噪声扰民；厂区东侧/北侧临近居民（东侧厂界距居民楼约3米）。第三方检测显示厂界昼间超标。
- **处理**：处罚 + 督促企业降噪整改 + 对举报人奖励 + 化解舆情。
- **评查启示**：
  - 噪声案证据：委托有资质第三方检测、厂界布点（距敏感点距离须记录）、昼/夜时段明确。
  - 体现"行政监管 + 社会监督"协同与"处罚+整改+奖励"闭环，是同类投诉处理示范。


## 六、中央生态环保督察典型案例（评查线索）

### 案例8：辽宁葫芦岛市生活污水收集处理短板明显（第三轮第六批督察通报）
- **评查启示**：生活污水收集管网、污水厂溢流、进水浓度低等系督察高频问题；地方案卷评查应对该类案件查办深度、整改闭环重点核查。


## 七、败诉/风险要点速记（本案集共性）

1. **采样/监测程序不合法** → 败诉首要风险（OBD作弊、VOCs篡改均依赖监测证据链）。
2. **证据链不完整** → 单一证据定案（如仅笔录无实物/视频/鉴定）。
3. **行刑衔接文书缺失** → 移送类案卷须含移送函、鉴定、量化达标证明。
4. **程序告知不全** → 复议范围扩大后告知内容须同步升级。
5. **一罚了之** → 缺少整改指导/后督察，与督察"反对以问责代替整改"导向相悖。
