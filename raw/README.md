# raw/ 只读证据层

## 概述

本目录存放原始证据资料，只读，禁止 AI 直接修改。

## 目录结构

```
raw/
├── laws/           # 法律文件
├── standards/      # 标准文件
├── reports/        # 报告文件
├── datasets/       # 数据集
└── README.md       # 本文件
```

## 入仓规范

1. 文件必须是原始证据，未经 AI 修改
2. 文件名格式：`{来源}-{日期}-{主题}.{扩展名}`
3. 每个文件必须包含元数据注释

## 元数据格式

```markdown
---
title: "文件标题"
source: "来源"
date: "2026-01-01"
version: "1.0"
industry: "行业标识"
---
```

## 注意事项

- 入仓前确认文件真实性和完整性
- 保留原始文件格式，不转换为 Markdown
- 定期备份，防止数据丢失
