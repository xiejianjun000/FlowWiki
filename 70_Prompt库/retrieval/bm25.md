# BM25 Retrieval Prompt

## 任务

使用 BM25 算法进行全文检索。

## 输入

- 查询词
- 文档集合
- 参数 k1, b

## 输出

- 排序后的文档列表
- BM25 评分

## 公式

```
score(D, Q) = sum(IDF(qi) * (f(qi, D) * (k1 + 1)) / (f(qi, D) + k1 * (1 - b + b * |D|/avgdl)))
```

## 约束

- 默认 k1=1.2, b=0.75
- 支持中文分词
- 返回 top-k 结果
