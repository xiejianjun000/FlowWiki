# 📮 送达合规审查（方式·回证·期限预警）

## 一、审查目标

确保执法文书的送达行为符合法定形式、程序正当、期限合规，且送达回证能够完整、准确地反映送达全过程，避免因送达瑕疵导致程序违法或行政行为被撤销。

## 二、审查依据

- 《中华人民共和国行政处罚法》第61条
- 《中华人民共和国民事诉讼法》第87条至第95条
- 《生态环境行政处罚办法》第58条
- 《最高人民法院关于适用〈中华人民共和国民事诉讼法〉的解释》第130条至第141条

## 三、审查要点

### 3.1 送达方式选择与适用顺序

| 送达方式 | 适用条件 | 合规要求 |
|---------|---------|---------|
| **直接送达** | 当事人或其同住成年家属、委托代理人、指定代收人 | 优先适用；应注明送达地点、时间、收件人身份及与当事人关系 |
| **留置送达** | 当事人拒绝签收或无法直接送达 | 需有见证人（基层组织代表或公证人员）并在回证上注明拒收事由、日期；全程录音录像可作为辅助证据 |
| **邮寄送达** | 直接送达有困难 | 必须使用**挂号信**或**EMS**（不得使用普通快递）；以回执上注明的收件日期为送达日期 |
| **公告送达** | 受送达人下落不明或用其他方式无法送达 | 须在**省级以上公开发行的报纸**或**生态环境部门门户网站**公告；公告期为**30日**（行政处罚决定书）或**60日**（其他文书） |
| **电子送达** | 当事人同意并确认电子送达地址 | 需有当事人书面同意或电子签名确认；送达凭证应保存完整 |

> ⚠️ **审查红线**：不得在未尝试直接送达的情况下直接采用公告送达；不得以普通快递替代邮寄送达。

### 3.2 送达回证审查要素

送达回证是证明送达行为合法性的核心证据，审查时应逐项核对：

| 审查项 | 合规标准 | 常见问题 |
|--------|---------|---------|
| 文书名称与文号 | 与送达文书一致 | 回证上文书名称与实际送达文书不符 |
| 送达时间 | 精确到日（必要时精确到时） | 时间空白或模糊 |
| 送达地点 | 具体到门牌号、房间号 | 仅写“单位”或“家中” |
| 收件人签名/盖章 | 当事人、同住成年家属、代理人签名 | 无签名或代签无授权 |
| 见证人信息 | 留置送达时必须有见证人签名及单位盖章 | 见证人身份不明或未签名 |
| 送达人签名 | 两名以上执法人员签名 | 仅一名执法人员签名 |
| 拒收事由记载 | 留置送达时须详细记载拒收原因 | 仅写“拒收”无具体事由 |
| 备注栏 | 如有补充说明应完整记录 | 备注空白但实际有特殊情况 |

### 3.3 期限预警与时效审查

| 文书类型 | 法定送达期限 | 预警阈值 |
|---------|-------------|---------|
| 行政处罚决定书 | 作出后**7日内**送达 | 第5日触发黄色预警 |
| 听证告知书 | 听证**7日前**送达 | 第5日触发黄色预警 |
| 责令改正通知书 | 一般无明确法定期限，但应合理及时 | 超过15日触发红色预警 |
| 查封/扣押决定书 | **当场**或**24小时内**送达 | 超过24小时触发红色预警 |

> 📌 **审查提示**：案卷评查时应将送达日期与文书作出日期进行比对，确认是否存在超期送达情形。对于邮寄送达，以**回执签收日期**为准，而非寄出日期。

### 3.4 特殊情形审查

| 情形 | 审查要点 |
|------|---------|
| 当事人为法人或其他组织 | 应送达至法定代表人、主要负责人或负责收件的人；不得仅送达至门卫或前台（除非有授权） |
| 当事人为自然人 | 本人不在时，可交其**同住成年家属**签收，但应在回证上注明关系 |
| 当事人下落不明 | 需有公安机关或社区出具的证明，方可启动公告送达程序 |
| 委托送达 | 需有委托函及受托机关的回证；受托机关应在**15日内**完成送达 |
| 涉外
