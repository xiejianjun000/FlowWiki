# 法律适用正确性审查

> 本 playbook 依据《生态环境行政执法案卷评查细则（2024年版）》第3.2节"法律适用正确性"审查标准编制。
> 审查目标：确保执法决定援引的法律、法规、规章条款**准确、完整、有效**，且**与违法事实相对应**。

---

## 一、审查要点清单

### 1.1 法条引用准确性
- [ ] 引用的法律文件名称是否完整、规范（全称+文号）
- [ ] 引用的条款层级是否正确（条、款、项、目）
- [ ] 引用的条款内容是否与违法事实对应
- [ ] 是否引用已废止或修改前的条款

### 1.2 法律适用完整性
- [ ] 是否同时引用**行为定性条款**和**处罚裁量条款**
- [ ] 涉及多个违法行为的，是否分别引用对应条款
- [ ] 涉及从重、从轻、减轻或不予处罚的，是否引用相应依据
- [ ] 涉及责令改正的，是否引用责令改正的法律依据

### 1.3 法律效力与时效
- [ ] 所引法律文件在违法行为发生时是否**现行有效**
- [ ] 所引法律文件在作出处罚决定时是否**现行有效**
- [ ] 是否适用**从旧兼从轻**原则（《立法法》第93条）
- [ ] 新旧法过渡期条款是否被正确适用

### 1.4 法律竞合处理
- [ ] 是否存在同一行为同时违反多个法律规范的情形
- [ ] 是否按**特别法优于一般法**、**新法优于旧法**原则处理
- [ ] 是否按**上位法优于下位法**原则审查
- [ ] 地方性法规与部门规章冲突时，是否按法定程序处理

## 二、常见错误类型与示例

### 2.1 引用废止/失效法律
| 错误示例 | 正确做法 |
|---------|---------|
| 引用《环境保护法》（1989版） | 应引用《环境保护法》（2014修订，2015.1.1施行） |
| 引用《大气污染防治法》（2015版） | 应引用《大气污染防治法》（2018修正） |
| 引用《建设项目环境保护管理条例》（1998版） | 应引用《建设项目环境保护管理条例》（2017修订） |

### 2.2 条款层级引用错误
| 错误示例 | 正确做法 |
|---------|---------|
| "依据《水污染防治法》第83条"（未指明款） | "依据《水污染防治法》第83条第2款" |
| "依据《大气污染防治法》第99条"（未指明项） | "依据《大气污染防治法》第99条第1项" |
| 引用"第XX条"但该条无款/项划分 | 核实该条实际结构后准确引用 |

### 2.3 定性条款与裁量条款混淆
| 错误示例 | 正确做法 |
|---------|---------|
| 仅引用《大气污染防治法》第99条（裁量条款） | 同时引用第18条（禁止超标排污）和第99条（处罚） |
| 仅引用《水污染防治法》第10条（原则性条款） | 应引用具体禁止性条款+处罚条款 |
| 引用《环境保护法》第59条（按日计罚）但未引用基础违法行为条款 | 先引用基础违法条款，再引用按日计罚条款 |

### 2.4 法律竞合处理错误
| 情形 | 处理原则 | 示例 |
|------|---------|------|
| 同一行为违反《大气污染防治法》和《噪声污染防治法》 | 按特别法优先 | 噪声污染优先适用《噪声污染防治法》 |
| 同一行为违反《水污染防治法》和《环境保护法》 | 按特别法优先 | 水污染优先适用《水污染防治法》 |
| 地方规定与国家标准冲突 | 按上位法优先 | 地方标准不得低于国家标准 |
