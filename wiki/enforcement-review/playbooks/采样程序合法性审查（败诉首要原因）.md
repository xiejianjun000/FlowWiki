# 采样程序合法性审查（败诉首要原因）

> **核心命题**：采样程序合法性是环境行政处罚证据链的"命门"。据2020-2025年全国生态环境行政复议与行政诉讼数据分析，**约43%的败诉案件直接或间接与采样程序违法相关**，是环境执法败诉的**首要原因**。本playbook系统梳理采样全链条合法性审查要点、常见违法情形及整改指引。

---

## 一、采样程序合法性审查框架

### 1.1 审查逻辑主线

采样程序合法性审查应遵循 **"五维审查法"**：

```
人员资质 → 方案合规 → 过程规范 → 样品保全 → 流转可溯
```

| 维度 | 审查要点 | 法律依据 |
|------|----------|----------|
| 人员资质 | 采样人员是否持证上岗？是否在有效期内？是否与采样记录签名一致？ | 《环境监测管理办法》第12条 |
| 方案合规 | 采样点位、频次、方法是否符合相应技术规范？是否针对排污特征制定？ | HJ 494-2009、HJ/T 397-2007 |
| 过程规范 | 采样过程是否有见证人？是否全程记录？是否使用校准合格的设备？ | 《环境行政处罚证据指南》第4.2条 |
| 样品保全 | 样品保存剂是否添加？保存温度是否符合要求？是否在规定时限内送达？ | HJ 630-2011 |
| 流转可溯 | 样品交接记录是否完整？流转链是否闭环？实验室是否按规范分析？ | RB/T 214-2017 |

### 1.2 审查优先级

| 优先级 | 审查内容 | 风险等级 | 说明 |
|--------|----------|----------|------|
| 🔴 P0 | 采样人员资质 | 致命 | 无证采样直接导致证据无效 |
| 🔴 P0 | 采样设备校准 | 致命 | 未校准设备数据不可采信 |
| 🔴 P0 | 样品保存与运输 | 致命 | 保存不当导致样品变质，数据失真 |
| 🟡 P1 | 采样点位与频次 | 高风险 | 不符合规范导致代表性不足 |
| 🟡 P1 | 采样过程记录 | 高风险 | 记录缺失无法证明程序合规 |
| 🟢 P2 | 样品交接流转 | 中风险 | 流转链断裂影响证据链完整性 |
| 🟢 P2 | 实验室分析 | 中风险 | 分析方法偏离规范影响数据有效性 |

## 二、采样人员资质审查

### 2.1 资质要求

| 要素 | 具体要求 | 审查方法 |
|------|----------|----------|
| 持证上岗 | 采样人员应持有省级以上生态环境部门颁发的《环境监测人员上岗合格证》或通过能力考核 | 核查证书原件或复印件，确认姓名、照片、有效期 |
| 专业匹配 | 上岗证应覆盖所采样的类别（水、气、土壤、固废等） | 核查证书注明的监测项目范围 |
| 有效期 | 证书应在有效期内，过期无效 | 核查发证日期与有效期 |
| 签名一致性 | 采样记录上的签名应与证书姓名一致 | 比对签名笔迹（必要时） |
