# 程序违法排查 SOP

## 一、概述

本 SOP 用于系统排查生态环境执法案卷中存在的程序性违法问题。程序违法是案卷评查中的**一票否决项**，一旦确认，将导致行政处罚决定被撤销或确认违法。本 SOP 聚焦于 **告知五要素、法定期限、法制审核、集体讨论** 等核心程序环节。

## 二、排查清单

### 2.1 告知五要素排查

> 依据：《行政处罚法》第四十四条、第六十一条；《生态环境行政处罚办法》第五十四条、第五十五条。

**排查要点：**
- [ ] **告知主体**：是否以作出处罚决定的机关名义告知？
- [ ] **告知对象**：是否准确列明当事人（单位名称/个人姓名）？
- [ ] **告知事实**：是否清晰陈述违法事实、证据及认定依据？
- [ ] **告知理由**：是否说明拟作出处罚的理由（法律依据）？
- [ ] **告知权利**：是否告知当事人享有陈述、申辩权？是否告知申请听证的权利（符合听证条件的）？

**常见违法情形：**
- 告知书与处罚决定书由不同机关作出（如委托执法未注明委托机关）。
- 告知内容与最终处罚决定不一致（如告知罚款10万，决定罚款20万，未重新告知）。
- 未告知听证权利（对个人罚款>2000元、单位罚款>10000元、责令停产停业、吊销许可证等情形）。

### 2.2 法定期限排查

> 依据：《行政处罚法》第六十条；《生态环境行政处罚办法》第六十四条、第六十五条。

**排查要点：**
- [ ] **立案期限**：发现违法线索后，是否在**7日内**决定是否立案？（特殊情形可延长至15日）
- [ ] **调查取证期限**：立案后至作出处罚决定前，是否存在超期未结案？（一般案件**90日**，经批准可延长**30日**）
- [ ] **告知期限**：作出处罚决定前，是否在**合理期限**内（通常为**3日**以上）送达告知书？
- [ ] **决定期限**：自立案之日起，是否在**90日**内作出处罚决定？（法律另有规定的除外）
- [ ] **送达期限**：处罚决定书是否在作出后**7日内**送达当事人？

**常见违法情形：**
- 立案超期未说明理由。
- 调查取证超期未办理延期审批。
- 告知书与处罚决定书同日送达（剥夺当事人陈述申辩权）。
- 处罚决定书作出后超过7日才送达。

### 2.3 法制审核排查

> 依据：《行政处罚法》第五十八条；《生态环境行政处罚办法》第六十六条。

**排查要点：**
- [ ] **审核范围**：是否属于应当进行法制审核的案件？（涉及重大公共利益、直接关系当事人重大权益、经过听证程序、案件情况疑难复杂涉及多个法律关系的）
- [ ] **审核主体**：是否由法制审核机构（或专职法制审核人员）进行审核？
- [ ] **审核内容**：是否对执法主体、执法程序、事实证据、法律适用、裁量基准等进行全面审核？
- [ ] **审核意见**：是否出具书面审核意见？审核意见是否明确（同意/补正/退回）？
- [ ] **审核记录**：审核意见是否附卷？是否由审核人员签名或盖章？

**常见违法情形：**
- 应当进行法制审核的案件未经审核即作出决定。
- 法制审核流于形式（审核意见仅写“同意”，无具体分析）。
- 审核人员不具备资格（如非公职律师、非法律专业且未经培训）。

### 2.4 集体讨论排查

> 依据：《行政处罚法》第五十七条第二款；《生态环境行政处罚办法》第六十七条。

**排查要点：**
- [ ] **讨论范围**：是否属于应当集体讨论的案件？（情节复杂、重大违法行为、拟给予较重处罚的）
- [ ] **讨论主体**：是否由**行政机关负责人**（正职或分管副职）集体讨论决定？
- [ ] **讨论记录**：是否有完整的集体讨论记录？记录是否载明时间、地点、主持人、参加人、讨论内容、表决意见及结论？
- [ ] **讨论结论**：讨论结论是否与处罚决定一致？是否存在“先决定后讨论”的情形？

**常见违法情形：**
- 集体讨论记录缺失或过于简单（仅写“经讨论，同意处罚”）。
- 参加讨论人员不符合法定要求（如仅有科室负责人，无机关负责人）。
- 讨论时间晚于处罚决定作出时间（程序倒置）。
