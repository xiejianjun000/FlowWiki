# 法律适用正确性审查

> 法律适用正确性审查是案卷评查的核心环节，直接决定执法行为的合法性基础。本概念页系统梳理审查要点、常见错误类型及评查标准。

## 一、审查原则

### 1.1 法定原则
- 处罚必须有明确的法律、法规或规章依据
- 不得以规范性文件、内部规定或会议纪要作为处罚依据
- 上位法优于下位法，特别法优于一般法，新法优于旧法

### 1.2 准确原则
- 引用的法条必须与违法行为性质、情节相匹配
- 法条引用应具体到条、款、项、目
- 不得张冠李戴、断章取义

### 1.3 完整原则
- 定性依据与处罚依据应分别引用，不得混淆
- 涉及多个违法行为的，应逐一列明法律依据
- 引用法条应包含完整的法律名称和条文序号

### 1.4 时效原则
- 适用行为发生时的有效法律（从旧兼从轻原则）
- 新法施行后，不得再引用已被废止或修改的法条
- 注意法律修订过渡期的适用规则

## 二、审查要点

### 2.1 法律依据的存在性
- [ ] 处罚决定是否明确引用了法律依据
- [ ] 所引法律、法规、规章是否现行有效
- [ ] 是否存在以“有关规定”等模糊表述代替具体法条的情形

### 2.2 法律依据的层级性
- [ ] 是否适用了正确的法律层级（法律→行政法规→地方性法规→规章）
- [ ] 是否存在下位法超越上位法设定处罚的情形
- [ ] 地方性法规或规章是否在法定权限范围内

### 2.3 法条引用的准确性
- [ ] 引用的条、款、项、目是否与违法行为对应
- [ ] 定性条款与处罚条款是否分别引用且逻辑一致
- [ ] 是否存在引用已废止、被修改或未生效法条的情形

### 2.4 法律适用的完整性
- [ ] 是否遗漏了应当适用的法条（如从轻、减轻、不予处罚条款）
- [ ] 涉及竞合违法行为的，是否依法选择适用
- [ ] 是否遗漏了程序性法律依据（如听证、告知等）

### 2.5 新旧法衔接
- [ ] 违法行为发生时间与法律生效时间是否匹配
- [ ] 是否正确适用从旧兼从轻原则
- [ ] 过渡期条款是否被正确引用

## 三、常见错误类型

### 3.1 法律依据缺失
- 处罚决定书未引用任何法律依据
- 仅引用“根据有关规定”或“依据相关法律”
- 引用内部文件、会议纪要、领导批示作为依据

### 3.2 法律依据错误
- 引用已废止的法律（如旧《大气污染防治法》）
- 引用未生效的法律（如尚未施行的修正案）
- 引用不存在的法条（如编造条文序号）

### 3.3 法条引用不准确
- 只引条不引款、项，导致适用对象不明确
- 定性依据与处罚依据混淆（如将定性条款当作处罚条款引用）
- 引用法条与违法行为性质不符（如用《水污染防治法》处罚大气污染）

### 3.4 法律层级错误
- 用部门规章代替法律法规设定处罚
- 地方性法规超越权限设定行政处罚
- 下位法与上位法冲突时仍适用下位法

### 3.5 遗漏适用
- 未引用从轻、减轻或不予处罚的法定情节
- 未引用责令改正的法律依据
- 未引用告知、听证等程序性法律依据

### 3.6 新旧法适用错误
- 新法施行后仍引用旧法
- 未正确适用从旧兼从轻原则
- 对过渡期条款理解错误

## 四、评查标准（依据《评查细则》第2.3节）

### 4.1 合格标准
- 处罚决定书明确引用法律、法规或规章
- 引用的法条现行有效
- 法条引用具体到条、款、项
- 定性依据与处罚依据分别引用且逻辑一致
- 法律适用与违法行为性质、情节相匹配

### 4.2 不合格情形（扣分项）
- 未引用法律依据（扣全部分值）
- 引用
