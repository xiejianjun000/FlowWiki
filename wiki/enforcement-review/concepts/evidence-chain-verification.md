# 证据链闭环核验 SOP

## 一、核验目标

确保案卷中用于认定违法事实的全部证据之间能够相互印证、形成完整的证明体系，排除合理怀疑，避免仅凭单一证据或孤证定案。核验的核心是检验证据的 **关联性**、**合法性**、**真实性** 以及 **逻辑闭环性**。

## 二、核验原则

1.  **孤证不能定案原则**：单一证据（尤其是当事人陈述、单一证人证言、无旁证的现场检查记录）不能作为认定案件事实的唯一依据。
2.  **印证证明原则**：不同种类的证据（如书证、物证、视听资料、电子数据、证人证言、当事人陈述、鉴定意见、勘验笔录、现场笔录）之间应能相互印证，指向同一待证事实。
3.  **排除合理怀疑原则**：全案证据形成的证明链条必须足以排除对主要事实认定的合理怀疑。存在无法排除的矛盾点时，应作出有利于当事人的认定。
4.  **程序合法性原则**：证据的收集、固定、保管、移送等程序必须符合法定要求，否则不得作为定案依据。

## 三、核验步骤（四步法）

### 第一步：证据清单梳理与分类

-   **动作**：调取案卷中的《证据清单》或自行梳理全部证据材料。
-   **要点**：
    -   列出所有证据，按种类分类（书证、物证、视听资料、电子数据、证人证言、当事人陈述、鉴定意见、勘验笔录、现场笔录）。
    -   标注每份证据的来源、收集时间、收集人、证明对象。
    -   **输出**：形成《案卷证据目录》。

### 第二步：单一证据合法性、真实性核验

-   **动作**：对每份证据进行独立审查。
-   **要点**：
    -   **合法性**：收集主体是否适格？程序是否合规？（如：现场检查是否双人执法并出示证件？抽样是否规范？鉴定机构是否具备资质？）
    -   **真实性**：是否为原件/原物？复制件/复制品是否与原件/原物相符？是否存在伪造、变造痕迹？电子数据是否经过哈希校验？
    -   **关联性**：该证据与待证的违法事实是否存在客观联系？
-   **输出**：对每份证据作出“合法/不合法”、“真实/不真实”、“有关联/无关联”的初步判断。

### 第三步：证据链闭环性核验（核心）

-   **动作**：将全部证据视为一个整体，检验其逻辑链条是否完整、闭合。
-   **核验维度**：

    | 核验维度 | 核验内容 | 常见问题（扣分项） |
    | :--- | :--- | :--- |
    | **1. 主体闭环** | 证据能否锁定违法主体？ | 当事人身份信息不清；现场检查笔录未记录当事人信息；证据指向主体与处罚决定主体不一致。 |
    | **2. 行为闭环** | 证据能否完整证明违法行为的全过程（时间、地点、方式、手段、情节）？ | 缺少关键时间节点证据；行为地点描述模糊；未证明行为持续状态。 |
    | **3. 结果闭环** | 证据能否证明违法行为造成的后果（污染程度、损害范围、违法所得等）？ | 缺少监测报告或监测报告无效；未核算违法所得；损害后果无量化证据。 |
    | **4. 因果闭环** | 证据能否证明违法行为与危害后果之间的因果关系？ | 仅有监测数据超标，但未排除其他污染源影响；未证明排放行为与超标结果之间的直接关联。 |
    | **5. 程序闭环** | 证据能否证明执法程序的合法性（立案、调查、告知、听证、决定、送达）？ | 缺少立案审批表；告知书送达回证缺失；听证笔录不完整；处罚决定书送达方式不合法。 |
    | **6. 逻辑闭环** | 证据之间是否存在无法解释的矛盾？ | 当事人陈述与现场检查笔录矛盾，且无其他证据佐证；不同证人证言之间冲突；监测数据与现场情况不符。 |

-   **核验方法**：
    -   **时间轴法**：将全部证据按时间顺序排列，检查时间线是否连续、无断裂。
    -   **证据链图法**：绘制证据关系图，明确每份证据证明的事实节点，检查节点之间是否连接成环。
    -   **矛盾点排查法**：逐一比对不同证据对同一事实的描述，查找矛盾点。对矛盾点，需审查是否有合理解释或补强证据。
