# 案卷评查实务经验更新

> 本页面汇集2025-2026年度案卷评查实务中的最新经验、常见误区与改进建议，供评查人员参考与更新认知。

## 一、评查理念更新

### 1.1 从“形式合规”到“实质合法+程序正当”

- **旧倾向**：过度关注文书格式、签字盖章、页码编号等形式要件，忽略实体与程序合法性。
- **新要求**：评查应同时关注：
  - 事实认定是否清楚、证据是否确实充分
  - 法律适用是否准确、裁量是否合理
  - 程序是否合法、当事人权利是否保障
- **实务要点**：形式瑕疵可补正，实体与程序违法不可补正。

### 1.2 从“扣分导向”到“提升导向”

- **旧倾向**：评查以扣分为主，忽视反馈与改进。
- **新要求**：评查结果应形成改进建议，帮助执法单位提升案卷质量。
- **实务要点**：评查报告应包含“亮点”与“改进建议”两部分。

## 二、常见误区与纠正

### 2.1 证据部分

| 误区 | 纠正 |
|------|------|
| 仅凭当事人陈述定案 | 必须有客观证据支撑，如现场检查记录、监测报告、照片视频等 |
| 证据未注明来源、提取时间 | 所有证据必须注明来源、提取时间、提取人 |
| 证据链不完整 | 证据应形成闭环，排除合理怀疑 |
| 忽略电子证据的合法性 | 电子证据应注明提取方式、存储介质、提取人签名 |

### 2.2 程序部分

| 误区 | 纠正 |
|------|------|
| 立案前调查取证 | 立案前可进行初步核查，但不得采取强制措施 |
| 告知程序流于形式 | 告知应明确、具体，告知当事人权利与救济途径 |
| 听证程序不规范 | 听证应依法组织，听证笔录应完整、准确 |
| 集体讨论记录缺失 | 重大案件必须集体讨论，记录应完整、签名齐全 |

### 2.3 文书部分

| 误区 | 纠正 |
|------|------|
| 文书编号不统一 | 文书编号应统一规则，便于追溯 |
| 日期填写错误 | 日期应准确，不得涂改 |
| 签名、盖章缺失 | 签名、盖章应齐全，不得代签 |
| 文书送达不规范 | 送达方式、时间、地点应记录清楚 |

## 三、2025-2026年新增评查要点

### 3.1 新法新规适用

- **《生态环境行政处罚办法》（2025年修订）**：新增“电子送达”“远程听证”等规定，评查时应关注是否适用。
- **《行政强制法》修订（2026年）**：新增“行政强制措施实施程序”细化规定，评查时应关注是否合规。

### 3.2 新类型案件评查

- **碳排放权交易案件**：关注配额核定、交易记录、核查报告等证据。
- **生态环境损害赔偿案件**：关注磋商协议、修复方案、资金使用等环节。
- **自动监测数据案件**：关注数据有效性、异常数据标记、运维记录等。

### 3.3 信息化评查工具

- **案卷评查系统**：支持在线评查、自动扣分、生成报告。
- **AI辅助评查**：可识别常见文书错误、证据缺失等问题。
- **电子档案管理**：案卷应同步生成电子档案，便于检索与共享。

## 四、评查实务技巧

### 4.1 评查前准备

- 熟悉最新法律法规与评查标准
- 了解案件类型与执法背景
- 准备评查工具（评查表、法规汇编、典型案例）

### 4.2 评查中要点

- **逐项核对**：对照评查标准逐项检查，避免遗漏
- **重点审查**：关注证据、程序、法律适用等关键环节
- **记录问题**：及时记录问题与疑点，便于后续讨论

### 4.3 评查后反馈

- 形成书面评查报告
- 提出具体改进建议
- 跟踪整改落实情况

## 五、典型案例与经验

### 5.1 案例一：证据链不完整导致案件撤销

- **改进**：评查时应重点关注证据链完整性。
