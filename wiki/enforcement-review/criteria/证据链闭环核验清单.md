# 📋 证据链闭环核验清单

> 本清单用于案卷评查中对证据链的**完整性、关联性、闭环性**进行逐项核验。  
> 核验目标：确保每一违法事实均有证据支撑，每一证据均能指向违法主体与行为，证据之间无断裂、无矛盾、无遗漏。

---

## 一、核验原则

1. **闭环原则**：证据链从“违法线索 → 违法主体 → 违法行为 → 违法后果 → 法律适用”形成完整闭环，不得出现断裂。
2. **关联原则**：每一证据必须与案件事实具有直接或间接关联，排除无关证据。
3. **充分原则**：证据数量与质量足以支撑认定违法事实，排除合理怀疑。
4. **合法原则**：证据收集、固定、保管、移送等环节符合法定程序。

---

### 2.1 证据来源与合法性核验

| 序号 | 核验项 | 核验标准 | 通过 | 不通过 | 备注 |
|------|--------|----------|------|--------|------|
| 1.1 | 证据来源是否明确 | 每一证据均注明来源（现场检查、监测报告、询问笔录、书证等） | □ | □ | |
| 1.2 | 证据收集程序是否合法 | 检查、采样、询问、查封扣押等程序符合法定要求 | □ | □ | |
| 1.3 | 证据是否经当事人确认 | 现场检查笔录、询问笔录等需当事人签字确认（拒绝签字的注明原因） | □ | □ | |
| 1.4 | 证据是否经执法人员签名 | 所有证据材料均有执法人员签名或盖章 | □ | □ | |
| 1.5 | 证据是否在法定期限内提交 | 证据材料在法定期限内提交至案卷 | □ | □ | |

| 序号 | 核验项 | 核验标准 | 通过 | 不通过 | 备注 |
|------|--------|----------|------|--------|------|

| 2.2 | 证据是否指向违法行为 | 证据能够证明违法行为的存在、时间、地点、方式 | □ | □ | |
| 2.3 | 证据是否指向违法后果 | 证据能够证明违法后果（污染、损害、风险等） | □ | □ | |
| 2.4 | 证据之间是否相互印证 | 不同证据之间无矛盾，能够相互支撑 | □ | □ | |
| 2.5 | 证据是否排除其他可能性 | 证据能够排除合理怀疑，形成唯一结论 | □ | □ | |
