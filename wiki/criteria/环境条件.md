# 环境条件判据集

## 判据数量

4 条

## 判据列表

### 1. 系统负载

- **描述**：CPU/内存/磁盘使用率在正常范围
- **阈值**：CPU < 80%, 内存 < 85%, 磁盘 < 90%
- **来源**：运维基线

### 2. 网络连通性

- **描述**：关键网络链路正常
- **阈值**：延迟 < 100ms, 丢包率 < 0.1%
- **来源**：运维基线

### 3. 服务可用性

- **描述**：关键服务正常运行
- **阈值**：可用率 > 99.9%
- **来源**：SLA 协议

### 4. 版本一致性

- **描述**：各组件版本匹配
- **阈值**：版本偏差 = 0
- **来源**：配置管理规范
