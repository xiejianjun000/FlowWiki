# 执法办案场景

## 概述

基于案卷评查标准，审查执法程序合法性和证据完整性。

## 场景列表

### 案卷评查
- **触发条件**：用户上传案卷或询问评查标准
- **调用 Skill**：程序审查、证据审核、法律适用检查
- **输出**：评查报告

### 程序合法性检查
- **触发条件**：用户询问执法程序
- **调用 Skill**：步骤验证、时限计算、文书规范
- **输出**：程序合法性分析报告
