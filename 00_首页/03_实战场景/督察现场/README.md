# 督察现场场景

## 概述

为生态环境保护督察提供现场支撑，包括问题研判、证据固定和整改建议生成。

## 场景列表

### 现场问题研判
- **触发条件**：用户描述督察现场发现的问题
- **调用 Skill**：风险识别、判据匹配、法规检索
- **输出**：问题定性分析报告（违规类型、严重程度、法律依据）

### 证据固定指引
- **触发条件**：用户询问现场取证要求
- **调用 Skill**：证据审核、文书规范
- **输出**：证据固定清单（取证要素、文书模板、链路要求）

### 整改建议生成
- **触发条件**：用户要求生成整改方案
- **调用 Skill**：合规建议、影响分析
- **输出**：整改建议书（整改措施、时限要求、验收标准）
