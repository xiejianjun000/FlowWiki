# 根因分析场景

基于数据溯源和异常检测技术，研判问题根因和影响范围。

## 场景说明

为数据分析团队提供根因定位支撑，包括异常检测、趋势分析和跨域追踪。

## 典型场景

### 根因定位

- **触发条件**：用户描述异常现象或提交分析请求
- **输入**：异常描述、相关数据
- **输出**：根因分析报告

### 趋势分析

- **触发条件**：用户询问数据变化趋势
- **输入**：时间序列数据
- **输出**：趋势分析报告

- 环境条件判据集
