# 排污许可场景

## 概述

基于排污许可证和环评文件，进行合规审查和达标判定。

## 场景列表

### 环评文件审查
- **触发条件**：用户上传环评文件
- **调用 Skill**：合规审查、标准匹配、公众参与核查
- **输出**：审查意见

### 排污许可证分析
- **触发条件**：用户上传许可证或询问许可事项
- **调用 Skill**：许可证拆解、达标判定、变更评估
- **输出**：分析报告
