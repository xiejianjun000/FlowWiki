# 现场核查场景

为审计和核查部门提供现场支撑，包括问题研判、证据固定和整改建议生成。

## 场景说明

基于审计法和内部审计准则，对现场核查发现的问题进行研判、取证和整改。

## 典型场景

### 现场问题研判

- **触发条件**：用户描述现场发现的问题
- **输入**：问题描述
- **输出**：问题研判报告

### 证据固定指引

- **触发条件**：用户询问现场取证要求
- **输入**：核查场景
- **输出**：证据固定指引

### 整改建议生成

- **触发条件**：用户要求生成整改方案
- **输入**：问题清单
- **输出**：整改建议方案
