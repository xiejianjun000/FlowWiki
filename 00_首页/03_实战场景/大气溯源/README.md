# 大气溯源场景

## 概述

基于大气环境监测数据，研判污染来源和传输路径。

## 场景列表

### 溯源研判
- **触发条件**：用户上传监测数据或描述污染过程
- **调用 Skill**：判据匹配、区域传输分析、源类指纹识别
- **输出**：污染来源分析报告

### EKMA敏感性分析
- **触发条件**：用户询问O3污染成因
- **调用 Skill**：O3敏感性诊断、EKMA曲线分析
- **输出**：敏感性诊断报告
