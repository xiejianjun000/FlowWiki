# 审计准备场景

为企业或机构提供审计准备工作支撑，包括合规自查、材料准备和问题预判。

## 场景说明

基于审计法和企业内部控制基本规范，帮助企业做好审计前的准备工作。

## 典型场景

### 合规自查清单生成

- **触发条件**：用户询问审计准备事项
- **输入**：企业基本信息
- **输出**：合规自查清单

### 审计材料准备

- **触发条件**：用户询问需要准备的材料
- **输入**：审计类型
- **输出**：材料清单

### 问题预判与风险排查

- **触发条件**：用户描述企业情况
- **输入**：企业运营情况
- **输出**：风险排查报告
