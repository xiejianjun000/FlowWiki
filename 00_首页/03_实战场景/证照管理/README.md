# 证照管理场景

基于行政许可法和行业准入规定，进行证照审查和合规判定。

## 场景说明

为审批部门和企业提供证照管理支撑，包括证照审查、许可证分析和变更评估。

## 典型场景

### 证照文件审查

- **触发条件**：用户上传证照文件
- **输入**：证照文件
- **输出**：合规审查报告

### 许可证分析

- **触发条件**：用户上传许可证或询问许可事项
- **输入**：许可证文件
- **输出**：许可证分析报告

## 相关判据

- 审批条件判据集
- 合规边界判据集
