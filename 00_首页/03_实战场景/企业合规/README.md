# 企业合规场景

## 概述

基于企业所属行业和业务，生成合规清单和追踪政策更新。

## 场景列表

### 合规清单生成
- **触发条件**：用户询问企业合规要求
- **调用 Skill**：法规检索、风险识别、清单编制
- **输出**：合规清单

### 政策追踪
- **触发条件**：用户询问政策更新
- **调用 Skill**：政策监控、影响分析、合规建议
- **输出**：政策更新报告
