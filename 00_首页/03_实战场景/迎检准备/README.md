# 迎检准备场景

## 概述

为企业或地方提供生态环境保护督察/检查的迎检准备支持，包括合规自查、材料准备和问题预判。

## 场景列表

### 合规自查清单生成
- **触发条件**：用户询问迎检准备事项
- **调用 Skill**：清单编制、法规检索、标准匹配
- **输出**：合规自查清单（检查项、法律依据、自查方法、整改建议）

### 迎检材料准备
- **触发条件**：用户询问需要准备的材料
- **调用 Skill**：文书规范、清单编制
- **输出**：材料清单（环评文件、排污许可证、监测报告、台账记录等）

### 问题预判与风险排查
- **触发条件**：用户描述企业/区域情况
- **调用 Skill**：风险识别、影响分析、合规建议
- **输出**：风险排查报告（风险点、严重程度、整改优先级、应对策略）

## 检查类型

### 中央生态环境保护督察
- 重点：政治站位、生态修复、群众反映
- 材料：汇报材料、整改方案、长效机制

### 省级生态环境保护督察
- 重点：政策落实、监管执法、问题整改
- 材料：自查报告、执法卷宗、监测数据

### 日常执法检查
- 重点：排污许可、污染防治设施、在线监控
- 材料：环评批复、许可证、运行台账、监测报告

### 专项检查
- 重点：特定污染物、特定行业、特定区域
- 材料：专项合规证明、治理方案、达标证明
