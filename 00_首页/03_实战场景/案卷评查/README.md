# 案卷评查场景

为执法监督部门提供案卷质量评查支撑，包括程序审查、证据链审核和法律适用检查。

## 场景说明

基于行政处罚法和行政执法程序规定，对执法案卷进行标准化评查。

## 典型场景

### 案卷程序审查

- **触发条件**：用户上传案卷材料
- **输入**：案卷文件
- **输出**：程序审查报告

### 证据链审核

- **触发条件**：用户询问证据完整性
- **输入**：证据材料
- **输出**：证据链审核报告

### 法律适用检查

- **触发条件**：用户询问处罚依据
- **输入**：处罚决定书
- **输出**：法律适用检查报告
