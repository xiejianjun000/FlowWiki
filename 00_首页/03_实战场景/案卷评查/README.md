# 案卷评查场景

## 概述

基于行政执法案卷评查标准，审查执法程序合法性、证据完整性和法律适用准确性。

## 场景列表

### 案卷程序审查
- **触发条件**：用户上传案卷材料或询问评查标准
- **调用 Skill**：程序审查、步骤验证、时限计算
- **输出**：程序合法性审查报告（程序节点、时限合规、文书规范）

### 证据链审核
- **触发条件**：用户询问证据完整性
- **调用 Skill**：证据审核、文书规范
- **输出**：证据链审核报告（证据种类、关联性、合法性、充分性）

### 法律适用检查
- **触发条件**：用户询问处罚依据
- **调用 Skill**：法规检索、法律适用
- **输出**：法律适用审查报告（适用条款、自由裁量、处罚幅度）

## 评查标准

### 程序合法性
1. 立案审批规范
2. 调查取证程序合法
3. 告知权利到位
4. 听证程序合规（如适用）
5. 处罚决定审批规范
6. 送达程序合法
7. 执行程序合规

### 证据完整性
1. 证据种类齐全（书证、物证、证人证言、笔录等）
2. 证据来源合法
3. 证据与事实关联
4. 证据链闭合

### 法律适用准确性
1. 适用法律法规正确
2. 处罚种类和幅度适当
3. 自由裁量权行使合理
4. 从轻/从重/减轻处罚有据
