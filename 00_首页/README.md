# FlowWiki 首页

## 概述

FlowWiki 是一个基于 Karpathy LLM Wiki 架构的知识复利系统，实现"越用越懂"。

## 核心功能

### 01_知识图谱
- 行业知识结构可视化
- 概念关联图谱
- 实体关系图

### 02_判据体系
- 行业判据集
- 判据匹配引擎
- 判据更新记录

### 03_实战场景
- 大气溯源场景
- 执法办案场景
- 排污许可场景
- 企业合规场景

### 04_进化学习
- ACE 反思循环
- A-MEM 记忆卡片
- 知识复利记录

### 05_采集记录
- 入仓审计日志
- 资料来源管理
- 版本历史

### 06_系统运维
- 知识库健康度
- 检索配置
- 备份恢复

## 架构概览

```
L1 知识编译层  →  raw/ + wiki/ + 00_首页/
L2 检索增强层  →  BM25→nano-graphrag→LightRAG
L3 Spec-Driven →  spec/ + openspec/
L4 Agent 记忆  →  A-MEM 卡片 + ACE 反思
L5 Skill 化    →  4 操作 + 行业专属
L6 多 agent    →  CLAUDE.md + AGENTS.md
L7 场景层      →  可插拔业务外壳
```

## 核心原则

- **骨肉分离**：L2-L6 通用骨架，L1+L7 行业内容
- **证据链完整**：所有知识可追溯到原始证据
- **ACE 反思**：错误知识不进 wiki
- **知识复利**：任务→知识→Skill 三元组
