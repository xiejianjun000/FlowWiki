# 02_判据体系

## 概述

行业判据集与判据匹配引擎，支撑实战场景中的合规判定、标准适用等决策。

## 内容

### 判据集

按场景组织的判据条目，每条判据包含：

- **判据编号**：唯一标识
- **判据描述**：判定规则
- **法律依据**：溯源到 raw/laws/
- **标准依据**：溯源到 raw/standards/
- **适用场景**：关联 L7 场景

### 判据匹配引擎

通过 `criteria-match` skill 自动匹配判据：

1. 输入案件/项目信息
2. 引擎匹配适用判据
3. 输出匹配结果（命中/未命中/需人工判断）

### 判据更新记录

判据的新增、修改、废止记录。

- 来源：`openspec/changes/` 变更治理
- 更新机制：法规标准变更时触发
