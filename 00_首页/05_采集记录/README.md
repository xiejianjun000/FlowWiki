# 05_采集记录

## 概述

入仓审计日志、资料来源管理和版本历史，确保证据链完整可追溯。

## 内容

### 入仓审计日志

每次 raw/ 入仓操作的完整记录：

- **时间**：入仓时间
- **文件**：raw/ 文件路径
- **来源**：资料出处
- **操作者**：人类 / AI agent
- **ACE 结果**：通过/待核/矛盾

### 资料来源管理

按类型分类的原始资料索引：

- **法律文件**：`raw/laws/`
- **标准文件**：`raw/standards/`
- **报告文件**：`raw/reports/`
- **数据集**：`raw/datasets/`

索引模板：`_templates/raw_index.md.j2`

### 版本历史

raw/ 文件的版本控制记录：

- Git commit 历史
- openspec/changes/ 变更记录
- wiki/log.md 操作日志
