# 知识图谱

> 行业关键实体及其关系。

## 实体类型

- **法规**：法律、法规、规章、规范性文件
- **标准**：国家标准、行业标准、国际标准
- **机构**：监管部门、审批机构、服务机构
- **概念**：核心术语、方法论、技术框架
- **流程**：审批流程、操作流程、监督流程

## 关系类型

- 引用（法规引用标准）
- 衍生（概念衍生自理论）
- 制约（流程受法规制约）
- 协同（多机构协同执行）

## Dataview 查询模板

```dataview
TABLE type, confidence, updated
FROM "wiki/concepts"
WHERE status = "reviewed"
SORT updated DESC
```
