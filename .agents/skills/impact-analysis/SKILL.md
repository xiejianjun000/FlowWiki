# Impact Analysis Skill

> 影响分析 — 评估变更或事件的潜在影响

## 功能

评估企业变更事项、政策更新或突发事件对合规状态的潜在影响。

## 输入

```json
{
  "event": "事件描述",
  "entity": "企业名称",
  "current_status": "当前合规状态",
  "scope": "分析范围"
}
```

## 输出

```json
{
  "impact_level": "高|中|低",
  "affected_areas": [
    {"area": "受影响领域", "impact": "影响描述", "severity": "high|medium|low"}
  ],
  "recommendations": ["应对建议"],
  "timeline": "影响时间线"
}
```

## 分析维度

1. **法律合规**：是否违反法规要求
2. **经营影响**：对业务运营的影响
3. **财务影响**：经济损失或成本增加
4. **声誉影响**：对企业形象的影响

## 约束

- 影响评估需基于事实和数据
- 需要领域专家确认分析结论
- 评估结果需定期复核
