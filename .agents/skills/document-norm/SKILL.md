# Document Norm Skill — 文书规范

## 功能

审查和规范行政执法文书的格式、内容和程序要求，确保文书合法有效。

## 输入

```json
{
  "document_type": "立案审批表|调查报告|处罚决定书|送达回证|其他",
  "content": "文书内容",
  "case_info": {
    "case_number": "案号",
    "parties": "当事人信息",
    "violation": "违法行为"
  }
}
```

## 输出

```json
{
  "status": "合规|需修改|不合规",
  "issues": [
    {
      "field": "问题字段",
      "problem": "问题描述",
      "requirement": "规范要求",
      "suggestion": "修改建议"
    }
  ],
  "completeness_check": {
    "required_fields": ["必填字段列表"],
    "missing_fields": ["缺失字段"],
    "format_issues": ["格式问题"]
  },
  "summary": "审查结论"
}
```

## 文书类型与规范

1. **立案审批表**：案件来源、立案理由、审批意见
2. **调查询问笔录**：询问人、被询问人、询问内容、签字确认
3. **现场检查笔录**：检查时间地点、检查内容、当事人确认
4. **调查报告**：事实认定、证据分析、处理建议
5. **处罚告知书**：违法事实、处罚依据、处罚内容、申诉权利
6. **处罚决定书**：事实、证据、法律依据、处罚决定、救济途径
7. **送达回证**：送达方式、送达时间、收件人签字

## 约束

- 审查依据《环境行政执法文书规范》
- 必填字段不得缺失
- 文书间逻辑关系必须一致（案号、当事人、事实描述）
- 签字盖章要求必须明确
