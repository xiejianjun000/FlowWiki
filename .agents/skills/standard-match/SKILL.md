# Standard Match Skill

> 标准匹配 — 根据企业信息匹配适用的标准

## 功能

根据企业所属行业、产品类型和经营规模，匹配适用的国家标准、行业标准和国际标准。

## 输入

```json
{
  "industry": "行业类型",
  "products": ["产品列表"],
  "scale": "企业规模"
}
```

## 输出

```json
{
  "matched_standards": [
    {
      "standard": "标准名称",
      "type": "国家标准|行业标准|国际标准",
      "applicable_items": ["适用项目"],
      "priority": "强制|推荐"
    }
  ],
  "unmatched_items": ["未找到适用标准的项目"]
}
```

## 审查要点

1. **行业优先**：先查行业标准，再查通用标准
2. 强制性标准优先于推荐性标准
3. 地方标准优先于国家标准（如有）

## 约束

- 未匹配的项目需明确提示
- 标准更新后需重新匹配
- 注意标准的适用范围限定
