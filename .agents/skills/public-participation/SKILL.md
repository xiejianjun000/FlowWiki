# Public Participation Skill

> 公众参与核查 — 审查公众参与程序的合规性

## 功能

审查建设项目或决策中公众参与程序的合规性，包括参与方式、公示要求、意见反馈处理。

## 输入

```json
{
  "project": "项目名称",
  "participation_docs": "公众参与文件",
  "review_criteria": ["审查标准"]
}
```

## 输出

```json
{
  "compliance": "合规|不合规|部分合规",
  "issues": [
    {"item": "问题项", "requirement": "法定要求", "actual": "实际情况"}
  ],
  "suggestions": ["改进建议"]
}
```

## 审查要点

1. 参与方式是否合法
2. 公示期限是否达标
3. 意见反馈是否妥善处理
4. 公众参与报告是否完整

## 约束

- 需对照法定公众参与要求
- 注意地方性法规的差异
- 核查需有原始文件支撑
