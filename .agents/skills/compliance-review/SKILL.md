# Compliance Review Skill — 合规审查

## 功能

审查环评文件是否符合法律法规和技术标准。

## 输入

```json
{
  "file_type": "环评报告书|环评报告表|登记表",
  "content": "文件内容",
  "industry": "行业类型"
}
```

## 输出

```json
{
  "status": "合规|不合规|需补充",
  "issues": [
    {
      "type": "法律|标准|程序",
      "severity": "高|中|低",
      "description": "问题描述",
      "reference": "引用条款"
    }
  ],
  "summary": "审查结论"
}
```

## 审查要点

1. 法律符合性
2. 标准适用性
3. 程序合法性
4. 公众参与
5. 环保措施

## 约束

- 引用具体法律条款和标准条文
- 区分强制性要求和推荐性要求
- 给出明确的修改建议
