# Time Limit Calc Skill — 时限计算

## 功能

计算行政执法各环节的法定时限，验证程序是否在法定期限内完成。

## 输入

```json
{
  "case_type": "行政处罚|行政强制|行政许可",
  "key_dates": [
    {
      "event": "事件名称",
      "date": "日期",
      "type": "start|end|intermediate"
    }
  ],
  "case_complexity": "一般|重大|复杂",
  "extensions": [
    {
      "reason": "延长理由",
      "approved_by": "审批人",
      "extended_days": 0
    }
  ]
}
```

## 输出

```json
{
  "status": "合规|超期|临界",
  "calculations": [
    {
      "stage": "阶段名称",
      "legal_limit": "法定时限",
      "legal_basis": "法律依据",
      "start_date": "起算日",
      "deadline": "截止日",
      "actual_completion": "实际完成日",
      "days_used": 0,
      "days_remaining": 0,
      "result": "合规|超期|临界"
    }
  ],
  "overdue_stages": ["超期阶段"],
  "warning_stages": ["即将到期阶段"],
  "summary": "时限计算结论"
}
```

## 行政处罚法定时限

| 阶段 | 法定时限 | 法律依据 |
|------|---------|---------|
| 立案审查 | 7 个工作日 | 行政处罚法第 54 条 |
| 调查取证 | 无明确上限（合理期限） | 行政处罚法第 47 条 |
| 案件审查 | 90 日（含延长） | 行政处罚办法第 50 条 |
| 处罚前告知 | 7 个工作日内决定 | 行政处罚法第 44-45 条 |
| 听证申请 | 告知后 5 个工作日 | 行政处罚法第 47 条 |
| 听证组织 | 申请后 10 个工作日 | 行政处罚法第 48 条 |
| 处罚决定 | 审查结束后 7 个工作日 | 行政处罚办法第 52 条 |
| 送达 | 7 日内 | 行政处罚法第 61 条 |
| 执行 | 15 日内缴纳罚款 | 行政处罚法第 66 条 |

## 计算规则

1. **工作日 vs 自然日**：区分工作日和自然日计算
2. **扣除时间**：听证、公告、鉴定等时间不计入
3. **延长条件**：重大复杂案件经批准可延长
4. **起算点**：从"次日"开始计算（不含当日）

## 约束

- 时限计算精确到日
- 引用具体法律条文
- 超期标注严重程度（超期 1-3 日为瑕疵，超期 3 日以上为违法）
- 节假日顺延规则需注明
