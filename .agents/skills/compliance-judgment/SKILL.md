# Compliance Judgment Skill

> 合规判定 — 判定企业行为是否符合法规要求

## 功能

根据输入数据和法规标准，判定企业行为是否合规，输出合规判定结果。

## 输入

```json
{
  "entity": "企业名称",
  "behavior": "待判定行为",
  "applicable_rules": ["适用法规/标准"],
  "evidence": "证据材料"
}
```

## 输出

```json
{
  "judgment": "合规|不合规|待确认",
  "violations": [
    {
      "rule": "违反的法规/标准",
      "item": "违规项",
      "severity": "严重|一般|轻微"
    }
  ],
  "recommendations": ["整改建议"]
}
```

## 审查要点

1. 对比实际行为与法规要求
2. 考虑证据的法律效力
3. 判断是否超过许可范围

## 约束

- 判定需引用具体法规条款
- 证据不足时标注"待确认"
- 不可替代法律意见
